## 사업 구조

### 1. 제품 라인업
- **Happier Label**: 파우치, 에코백, 티슈커버, 우산, 머그컵 (레터링/패턴 소품)
- **MNNS**: 키링, 마우스패드, 보조배터리, 폰케이스 (데스크/테크 소품)
- **PART PART**: 노트, 플래너, 스티커 (문구류)
- **시즌/리빙**: 캘린더, 다이어리, 러그, 실내화
- **콜라보/한정**: 제주맥주, 이니스프리 등 브랜드 콜라보
- **기프트세트**: 기프트박스 A/B, 선물 포장 서비스
- 가격대: 3,000원 ~ 52,000원

### 3. 주간 업무 루틴 (매주 반복)
- **매일**: 롤리 - 매출 데이터 확인, 통장 잔고, 현금흐름 장부 / 썸머 - 데일리 매출 시트 기입(5분) / 이정원 - 핫트랙스 매출 수기 입력
- **월요일**: 썸머 - 인스타 인사이트 + GA 데이터 정리 (2~3시간), 조이 - 6개 채널 주간 매출 정리 (2시간)
- **화요일 오전**: 전체 주간 피드백 회의 (조이/썸머 데이터 기반)
- **화요일 오후**: 조이 - 메타 광고 효율 분석 (1~2시간)

### 4. 인스타그램 (@ohlollyday)
- 콘텐츠: 릴스, 캐러셀, 단일이미지, 스토리하이라이트
- 제작: 소니(비주얼 제작, 주 3~4개) → 노션 업로드 → 썸머(캡션 작성) → 롤리 확인 → 업로드
- 2월부터 광고비 증액 (월 200만원 수준)
- 주요 도구: 구글시트(데이터), 노션(회의록), 링크트리(클릭 추적), GA(트래픽/매출)

### 5. 사용 도구
- **노션**: 업무 아카이빙, 기획 노트, 콘텐츠 캘린더, 회의록
- **슬랙**: 사내 메신저, 매장 어시스턴트 일일 후기
- **구글 스프레드시트**: 매출 데이터, 급여 정리, 현금흐름 장부
- **이카운트**: 매출 정산, 견적서, 재고 관리
- **카페24**: 자사몰 어드민
- **메타 광고 관리자**: 광고 효율 분석
- **피그마**: 상세페이지 기획안

## 업무 규칙
- 데이터 분석 시 인사이트와 근거를 함께 제시
- "그래서 뭘 해야 하는지" 액션 아이템 필수
- 채널 분석 시 매출뿐 아니라 수수료/원가 반영한 영업이익 기준 사용
- 인스타 콘텐츠 분석 시 도달(vanity metric)보다 전환(구매) 중심으로 판단
