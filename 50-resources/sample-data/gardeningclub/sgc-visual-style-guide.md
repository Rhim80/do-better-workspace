'서울가드닝클럽' 비주얼 스타일 가이드

1. 디자인 철학: "도시의 일상을 바꾸는 그린스페이스"
도심의 유휴 옥상을 공유정원으로 전환하는 그린 라이프스타일 기업으로, 전통적 가드닝 브랜드의 earthtone 감성을 의도적으로 탈피합니다. 미니멀 모던 베이스에 네온 그린과 오렌지라는 파격적 포인트 컬러를 결합하여, '도시형 정원'이라는 새로운 카테고리의 시각 언어를 구축합니다. 한나 아렌트의 '인간의 조건'에서 영감받은 "LABOR! WORK! ACTION!" 철학이 브랜드 세계관의 뼈대이며, 이는 자연과의 노동(Labor), 공간에 정체성을 부여하는 작업(Work), 도시 환경에 선한 영향을 미치는 행동(Action)을 뜻합니다.

2. 핵심 테마: 미니멀 모던 + 도시형 자연 (Minimal Modern + Urban Nature)
블랙/화이트 기반의 깔끔한 구조 위에, 식물과 도시가 공존하는 에너지를 시각적으로 표현합니다. 일반적인 가드닝 브랜드가 선택하는 세이지 그린이나 크래프트 텍스처가 아닌, 네온 그린(#42ff8b)과 시그널 오렌지(#ff8f33)를 포인트로 써서 젊고 도시적인 감성을 전달합니다. 정원이 있는 라이프스타일을 제안하되, 그것이 힙하고 접근 가능한 것임을 시각적으로 보여줍니다.
톤앤매너: 모던한, 도시적, 자연주의, 젊은, 라이프스타일, 커뮤니티

4. 타이포그래피 (Typography)
'모던 라이프스타일 브랜드'로서의 위치를 폰트 선택으로 강화합니다. 영문은 경량 산세리프, 한글은 깔끔한 고딕 계열로 통일합니다.

브랜드 폰트 (영문 메인): Raleway
이유: 얇고 우아한 산세리프로, 라이프스타일 브랜드의 세련된 감성을 전달합니다. 주요 섹션 제목, 영문 키워드, 네비게이션에 사용합니다.

브랜드 폰트 (영문 서브): Montserrat
이유: Raleway보다 약간 두꺼운 기하학적 산세리프로, 데이터나 숫자가 포함된 영역에서 가독성을 보완합니다.

브랜드 폰트 (한글 메인): Pretendard
이유: 현대적이고 중립적인 한글 서체. Raleway와 자연스러운 조화를 이루며, 다양한 weight에서 안정적입니다.

브랜드 폰트 (한글 베이스): Noto Sans Korean
이유: 시스템 폴백 및 본문 텍스트용. Pretendard가 로드되지 않는 환경에서의 일관성을 보장합니다.

위계 질서 (Hierarchy):
H1 (브랜드/캠페인 헤드라인): Raleway Bold (36-48px) - 대문자 또는 소문자 통일
H2 (섹션 제목): Raleway SemiBold (24-28px)
H3 (서브 제목): Pretendard SemiBold (18-20px)
Body (본문): Pretendard Regular (15-16px) - 행간 1.6배
Label/Meta (라벨): Montserrat Medium (12-14px) - 영문 라벨, 숫자 데이터
Caption: Pretendard Regular (11-12px, 미드 그레이 컬러)

* 폰트 스택: `font-family: Raleway, Pretendard, sans-serif` (주요 섹션), `font-family: Montserrat, Noto Sans Korean, sans-serif` (데이터/본문 섹션)
* 폰트 정보는 웹사이트 CSS @import에서 직접 확인된 값입니다.

5. 그래픽 언어 및 모티프 (Graphic Language & Motifs)

식물 이미지 중심:
브랜드의 핵심 비주얼은 실제 식물과 정원 공간 사진입니다. 일러스트나 추상 그래픽보다 리얼 포토그래피를 우선합니다. 옥상에서 내려다보는 도시 전경과 식물의 공존이 핵심 비주얼 모티프입니다.

도시-자연 대비:
콘크리트, 철재 구조물, 도시 스카이라인과 녹색 식물의 대비를 의도적으로 연출합니다. '자연이 도시로 침투하는' 느낌을 지향하며, 목가적이거나 전원적인 이미지를 피합니다.

미니멀 레이아웃:
넉넉한 여백과 단순한 그리드를 사용합니다. 라이트 그레이(#F6F6F8) 배경 위에 깔끔한 카드형 컴포넌트로 콘텐츠를 정리합니다. 복잡한 장식 요소 없이 콘텐츠와 이미지가 직접 말하게 합니다.

사진 스타일:
자연광 선호. 옥상 위 푸른 하늘, 식물의 생생한 녹색, 흙을 만지는 손 등 감각적 이미지. 과도한 후보정 없이 있는 그대로의 생동감을 전달합니다. 도시 배경이 함께 잡히는 앵글을 의도적으로 선택합니다.

6. 레이아웃 및 컴포넌트 (Layout & Components)

전체 레이아웃:
라이트 그레이(#F6F6F8) 배경 기반. 화이트 카드 + 넉넉한 여백으로 콘텐츠를 구분합니다. imweb 플랫폼 기반으로, 풀와이드 히어로 + 그리드형 콘텐츠 섹션 + 고정 헤더 구조입니다.

헤더 (Header):
고정형(fixed). 반응형으로 데스크톱/모바일 분기. 로고(이미지+텍스트 조합) + 메가 드롭다운 네비게이션. 드롭다운 배경은 다크(#222222)로 대비를 줍니다.

카드 (Cards):
배경: 화이트(#FFFFFF)
테두리: 1px #E7E7E7 또는 없음
border-radius: 작은 값 (4-8px 추정, 부드러운 모서리)
hover 시 이미지 1.1 스케일 확대 (0.3s ease transition)

버튼 (Buttons):
Primary Button: 오렌지(#FF8F33) 배경 + 화이트 텍스트
Primary Hover: 딥 오렌지(#F59B17) 배경
Secondary Button: 테두리만 있는 고스트 버튼
border-radius: 소프트 라운드 (4-8px)

비주얼 섹션:
슬라이드/갤러리형 히어로 이미지. 전체 화면 너비 활용. 옥상 정원 풍경이나 가드닝 활동 장면을 메인으로 배치합니다.

7. 아이콘 (Iconography)

스타일: 라인 또는 미니멀 솔리드 아이콘. 라이프스타일 브랜드답게 과도하게 기능적이지 않은, 감성적인 아이콘을 사용합니다. 식물, 물뿌리개, 흙, 태양 등 가드닝 관련 소재.
색상: 다크 차콜(#212121) 기본. 활성 상태에서 오렌지(#FF8F33) 또는 네온 그린(#42FF8B) 적용.

8. 인터랙션 (Micro-interactions)

Hover (이미지): scale(1.1) 확대 효과, 0.3s ease transition
Hover (버튼): 오렌지(#FF8F33) -> 딥 오렌지(#F59B17) 색상 전환
Hover (특수): 네온 그린(#42FF8B) 발광 효과 - 디지털 환경에서 도시적 에너지를 표현
Image Rendering: -webkit-optimize-contrast 적용, 선명한 이미지 렌더링 우선
Page Transition: 부드러운 스크롤 기반, 급격한 전환보다 자연스러운 흐름 선호

9. 브랜드 공간 원칙 (Space Design Principles)

3개 지점 (2026년 기준):
- 1유로프로젝트 (성동구) - 공유 옥상정원
- 핸드픽트호텔 (동작구) - 호텔 옥상 콜라보
- 파크먼트 (서대문구) - 공유 공간 옥상

공통 원칙: 도심 옥상이라는 물리적 제약 안에서 녹색 공간을 최대화. 콘크리트와 식물의 의도적 대비. 멤버십 기반 커뮤니티 공간으로서 편안하면서도 관리된 느낌 유지.

운영 형태: 시즌 멤버십 (3개월, 6회 프로그램, 1인 63만원), 기업 워크숍, 조경 컨설팅, 브랜드 팝업 정원

10. 브랜드 필라 (Brand Pillars)

서울가드닝클럽의 3개 브랜드 필라는 한나 아렌트의 'vita activa' 개념에서 도출되며, 모든 비주얼 커뮤니케이션의 방향성을 결정합니다:

LABOR - 자연과 연결되는 노동. 흙을 만지고 식물을 돌보는 원초적 행위의 가치.
WORK - 도시 공간에 정체성을 표현하는 작업. 유휴 옥상을 정원으로 전환하는 창조적 작업.
ACTION - 도시 환경과 공동체에 미치는 선한 영향력. 그린스페이스가 도시의 일상을 바꾸는 실천.

요약
이 스타일 가이드는 다크 차콜(#212121) + 라이트 그레이(#F6F6F8) 미니멀 베이스, Raleway + Pretendard 타이포그래피, 오렌지(#FF8F33) + 네온 그린(#42FF8B) 포인트 컬러를 중심으로, '도시에서 자연을 가꾸는 새로운 라이프스타일'이라는 브랜드 경험에 집중합니다. 전통 가드닝 브랜드의 earthtone 공식을 거부하고, 도시적이면서 젊은 그린 라이프스타일을 제안하는 것이 핵심입니다.
