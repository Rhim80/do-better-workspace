# 서울가드닝클럽 - 봄시즌 데이터 활용

> **시연용 가상 시나리오입니다.** 실제 서울가드닝클럽을 참고했으나, 데이터와 미션은 교육 목적으로 재구성되었습니다.
>
> **상황**: 2026년 3월 말, 봄시즌을 앞두고 밀려드는 업무
> **당신의 역할**: 서울가드닝클럽 팀원

## 회사 소개

**서울가드닝클럽**

- **대표**: 이가영
- **콘셉트**: 도심 유휴 옥상을 공유정원으로 전환하는 그린 라이프스타일 기업
- **핵심 키워드**: "도시의 자연"
- **지점**: 3개소 (1유로프로젝트/성동구, 핸드픽트호텔/동작구, 파크먼트/서대문구)
- **B2C**: 시즌 멤버십 (3개월, 6회 프로그램, 1인 63만원)
- **B2B**: 기업 가드닝 워크숍, 조경 컨설팅, 브랜드 팝업 정원 연출

## 미션 1: 데이터 기반 식재 설계

**상황**: IT 기업에서 옥상정원 의뢰가 왔다. 120m2, 예산 2,500만원, ESG 보고서 연계 희망.

> 대표님: "보통 이런 문의 오면 현장 가서 눈으로 보고, 머릿속으로 식물 떠올리고,
> PPT로 제안서 만들어서 보내잖아요. 그게 3~4일 걸리는데,
> 우리한테 3년치 데이터가 있으니까 이걸 써보자."

**보통이라면?** 현장답사 + 경험 기반 식물 선정 + PPT 작업 = **3~4일**
**Claude Code가 있다면?** 데이터 필터링 + 검증 + 제안서 생성 = **30분**

### Step 1: 조건에 맞는 식물 필터링

```
"@sgc_plant_database.csv 에서 옥상정원에 적합한 식물을 추천해줘.

조건:
- 옥상적합: 상
- 서울적합도: 4점 이상
- 관리난이도: 3 이하 (기업 옥상이라 전담 관리자가 없어요)
- 실패율: 15% 미만

분류별(관목/초화류/그라스/허브)로 정리하고,
개화시기 기준으로 사계절 꽃이 피는 조합을 제안해줘."
```

### Step 2: 예산 안에서 3가지 안 비교

```
"식재 예산을 1,200만원으로 잡고 (나머지는 시공+자재),
120m2 옥상정원 식재 안을 3가지 만들어줘.

- A안: 고객선호도 최우선
- B안: 비용효율 최우선
- C안: 유지관리 최소화

각 안별 총 비용, 종수, 관리난이도 평균, 예상 실패율,
사계절 개화 커버리지를 비교 테이블로."
```

### Step 3: 과거 성공 프로젝트로 검증 + 제안서 생성

```
"@sgc_projects_2023_2026.csv 에서 비슷한 프로젝트를 찾아줘.
조건: 기업 클라이언트, 면적 80~150m2, 만족도 4.5 이상.

이 프로젝트들에서 공통 사용 식물과 위의 추천안을 교차 검증하고,
결과를 B2B 제안서 형태(HTML)로 만들어줘.

포함 내용:
1. 서울가드닝클럽 소개 (2~3줄)
2. 유사 프로젝트 레퍼런스 3건 (프로젝트명, 면적, 주요 식물, 만족도)
3. 추천 식재안 + 비교 테이블
4. 예상 견적 (식재비 + 시공비 + 연간 유지관리)
5. 기대 효과 (ESG 연계, 직원 복지, 공간 가치)

브랜드 컬러: 그린(#2D5016) + 크림(#F5F0E8)"
```

여기서부터 자유롭게:
- "이 제안서를 이메일 발송용 3문단 요약으로 만들어줘"
- "견적서를 별도 테이블로 분리해줘"
- "연간 유지관리 계약 제안도 추가해줘"

## 미션 2: 인스타그램 콘텐츠 → 비즈니스 전환

**상황**: 봄시즌 멤버십 모집이 2주 남았다. 인스타 팔로워는 느는데 매출로 안 이어진다.

> 대표님: "우리 인스타 예쁘다는 얘기는 많이 듣는데,
> 정작 멤버십 문의는 안 늘어요.
> 뭘 올려야 사람들이 '나도 해보고 싶다'고 느끼는 건지 모르겠어요."

**보통이라면?** 감으로 콘텐츠 기획 + 올리고 반응 보고 = **반복 시행착오**
**Claude Code가 있다면?** 성과 데이터 분석 + 전환 패턴 발견 + 콘텐츠 캘린더 = **20분**

### Step 1: 뭐가 잘 되고 있는지부터

```
"@sgc_instagram_2602.csv 2월 인스타그램 성과를 분석해줘.

콘텐츠유형별(릴스/카루셀/피드/스토리) 평균 도달, ER, 저장률 비교.
주제별(정원일상/식물관리팁/프로젝트사례/멤버후기/시즌안내/B2B사례)로도 쪼개줘.
어떤 조합이 가장 반응이 좋은지."
```

### Step 2: '보는 것'과 '움직이는 것'의 차이

```
"도달 → 프로필방문 → 링크클릭 → 문의전환 퍼널을 분석해줘.

좋아요는 많은데 문의가 없는 콘텐츠 vs
좋아요는 적은데 문의로 이어진 콘텐츠,
이 둘의 차이가 뭐야?
주제, 포맷, 캡션길이, 해시태그수 중 문의전환에 영향을 주는 요인은?"
```

### Step 3: 분석에서 실행으로

```
"분석 결과 기반으로 봄시즌 멤버십 모집용
인스타 콘텐츠 캘린더 2주분을 만들어줘.

조건:
- 주 3회 게시
- 전환율이 높았던 주제와 포맷 조합 우선
- 각 게시물별 캡션 초안 + CTA 포함
- '도시에서 자연을 가꾸는 경험'이라는 톤 유지

마지막에 '왜 이 조합을 선택했는지' 데이터 근거도 한 줄씩 달아줘."
```

여기서부터 자유롭게:
- "이 캘린더를 팀 공유용 테이블로 정리해줘"
- "캡션을 좀 더 감성적으로 다듬어줘"
- "B2B 문의 유도용 콘텐츠도 따로 3개 기획해줘"
