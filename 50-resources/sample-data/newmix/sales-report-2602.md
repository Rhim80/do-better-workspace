## 1. 성장 원인 분해 (Variance Analysis)

| 요인 | 기여액 | 비중 | 해석 |
|------|--------|------|------|
| 수량 효과 | +31,922,000원 | **100.4%** | 더 많이 팔렸다 |
| 단가 효과 | -139,000원 | -0.4% | 단가 거의 변동 없음 |
| 믹스 효과 | 0원 | 0.0% | 카테고리 구성 변화 미미 |
| **총 차이** | **+31,783,000원** | 100% | |

**해석 (프레임워크 패턴)**: **수량+ 단가- (지표상 미세)** → 사실상 "수량 단독 성장"
- 가격 할인을 한 것이 아니라(-0.1%는 노이즈 수준), **순수하게 더 많은 손님이 더 많이 산 것**
- 단가가 그대로인데 수량이 47% 늘었다 → 매장의 캐파(처리량)에 압박이 가해졌을 가능성

**비중 변화**: 커피포 82.8% → 82.0%, MD상품 11.1% → 11.7%, 스낵 6.1% → 6.3%
- 비중은 거의 그대로 유지된 채, **모든 카테고리가 동반 성장**
- 성장의 80%는 커피포에서 발생 → 커피포 재고/SKU 관리가 가장 중요

### BCG 매트릭스 적용

```
                       MoM 성장률 (높음/낮음)
                높음 (+30%↑)        낮음
       ┌──────────────────────┬──────────────┐
   높  │  커피포 (Star)        │              │
   매  │  82% 비중 + 46% 성장   │              │
   출  ├──────────────────────┼──────────────┤
   비  │  MD상품, 스낵          │              │
   중  │  (Question Mark)      │              │
       │  10%대 비중 + 50%+ 성장 │              │
       └──────────────────────┴──────────────┘
```

- **Star: 커피포** — 매출 비중도 크고 성장도 큼. 핵심 효자
- **Question Mark: MD상품/스낵** — 비중은 작지만 성장률은 커피포보다 높음. 추가 SKU 투입 시 Star로 끌어올릴 잠재력

### 외국인 vs 한국인 분리 비교

| 구분 | 1월 매출 | 2월 매출 | MoM | 객단가 1월→2월 |
|------|----------|----------|-----|----------------|

**가장 중요한 시사점**:

1. **성장의 본질은 외국인 관광객 유입**
   - 외국인 매출 +69.7%, 한국인 매출 -22.9% → 매출 구성이 외국인 90%에 가까워지는 중
   - 일본·유럽·대만·홍콩·미국이 동시에 +80~115% → 단일 국가 이슈가 아닌 **방한 외국인 트래픽 전반의 상승**

2. **한국인 고객은 오히려 빠지고 있다**
   - 1월 570건 → 2월 439건 (-23%)
   - 객단가는 그대로(28,500원) → 한국인은 적게 사는 게 아니라 **덜 오는 것**
   - 가설: 매장이 외국인으로 붐비면서 한국인 회피 현상 / 성수 인근 동선 변화 / SNS 노출 감소

3. **외국인 객단가 상승(+8%)**
   - 외국인 한 명이 더 많이 사고 있다 → 입소문 + 추천 효과로 추정

## 6. So What — 성장의 질에 대한 판단

**좋은 신호**:
- 가격 인하 없이 수량으로만 매출 +47% → 마진 압박 없는 건강한 성장
- 4주 내내 안정적 일평균 → 일회성 이벤트가 아님
- 모든 카테고리 동반 성장 → 매장 자체가 "더 잘 팔리는 곳"이 됨
- TOP 10 집중도 하락 → 롱테일 SKU도 살아나는 중
- 외국인 객단가 상승 → 단순 트래픽이 아닌 인당 구매력도 상승

**경계 신호**:
- **한국인 -23%는 무시할 수 없다**. 외국인 의존도가 87%까지 올라간 매장은 환율·관광·항공편 등 외부 변수에 매우 취약
- 2월 일평균 4.1M원 = 영업일 기준 약 200건/일 → 매장 캐파(결제·재고·접객) 한계 점검 필요
- 커피포 단일 카테고리 의존(82%) → 오리지날·볶은쌀 두 SKU 품절 리스크가 매출에 직격

## 7. Now What — 액션 제안

### 즉시 (3월 1주차)
1. **재고 안전선 재설정**: 오리지날 10T(1,193개/월), 볶은쌀(694개), 바나나우유맛(502개) — 일본 관광객 시즌 패턴 고려해 발주 리드타임 +20% 버퍼 확보
2. **주말 운영 강화**: 일요일 매출이 평일의 1.7배. 인력·결제 동선 주말 강화 검토 (특히 일요일)
3. **한국인 고객 이탈 원인 진단**: 한국인 객단가는 동일하므로 "왜 덜 오는가"가 문제. 1월/2월 한국인 결제 시간대·요일 비교, 인근 핫플 변화 체크

### 단기 (3월~)
4. **외국인 트래픽 유지를 위한 채널 점검**: 일본·대만·홍콩 SNS에서 어디가 트래픽 진원지인지 추적 (인스타·샤오홍슈·라인블로그 등). 진원 채널이 식으면 매출 즉시 영향
5. **3월 벚꽃 시즌 대비**: 외국인 관광객 추가 유입 가능성 — 4월 매출 가설(2월 일평균 ×1.2~1.4배) 미리 시뮬레이션 후 인력·재고 의사결정
6. **MD상품/스낵 SKU 확장**: Question Mark 영역. 비중은 작아도 성장률은 커피포와 동급. 외국인 선호 SKU 추가 테스트 (귀국 선물 수요)

### 중기 (Q2)
7. **고객 구성 다변화 전략**: 한국인 비중 12%까지 떨어진 것은 리스크. 평일 한국인 타깃 메뉴/프로모션 별도 설계 (외국인 트래픽이 적은 화~목 평일에 집중)
8. **품절 리스크 매뉴얼**: 단일 SKU 의존도가 높으므로 "오리지날 결품 시 대체 동선" 사전 정의
