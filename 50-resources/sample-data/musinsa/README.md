## 미션 1: 인플루언서 협업 성과 리뷰

**질문**: "메가 인플루언서한테 큰돈 쓴 건 잘한 거였나?"

### Step 1: 전체 현황부터

```
"@musinsa_influencer_2602.csv 2월 인플루언서 협업 전체 현황 정리해줘.
총 몇 건 협업했고, 총 비용 얼마 썼고, 등급별로 어떻게 분포되어 있는지.
플랫폼별 비중도 같이."
```

### Step 2: 프레임워크를 더해서 깊이를 만듭니다

같은 데이터를 **분석 프레임워크** 관점으로 다시 봅니다.

```
"이 데이터를 @influencer-effectiveness-framework.md 관점으로 다시 봐줘.

등급별 Engagement Rate랑 비용 대비 전환 효율을 비교하고,
인플루언서 티어 매트릭스로 시각화해줘.
X축 비용, Y축 전환수, 버블 크기 노출수, 색상 등급별로."
```

### Step 3: 궁금한 걸 바로 물어봅니다

```
"카테고리별로 쪼개서 봐줘.
아우터에서는 어떤 등급이 효율 좋고, 스니커즈는 다를 수 있잖아"
```

여기서부터는 자유롭게:
- "29CM 쪽 인플루언서만 따로 보면?"
- "틱톡이랑 인스타 효율 차이는?"
- "비용 대비 저장수가 높은 인플루언서 TOP 10은?"

**질문**: "릴스를 밀고 있는데 진짜 효과가 있나?"

```

```
