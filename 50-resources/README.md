# 50 Resources

참고 자료와 리소스를 저장하는 공간입니다.

## 폴더 구조

```
50-resources/
├── README.md          (이 문서)
└── sample-data/       실습용 샘플 데이터셋
    └── README.md
```

## 샘플 데이터

교육용 샘플 데이터는 클라이언트별 브랜치에서 제공됩니다.
`main` 브랜치에서는 비어 있으며, `git checkout ax/[클라이언트명]`으로 전환하면 해당 데이터가 나타납니다.

## 특징

- **외부 자료**: 본인이 작성하지 않은 외부 자료
- **참고용**: 필요할 때 참고하는 자료
- **정리된 형태**: 쉽게 찾을 수 있도록 카테고리별 정리
