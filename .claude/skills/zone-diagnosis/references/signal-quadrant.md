# 4분면 신호 도식 라이브러리

> zone-diagnosis Step 2가 호출. 생활인구 추세 × 매출 추세 매트릭스.

## 4분면 매트릭스

```
                    매출 ↑
                     │
   ↓ 객단가 상승      │      ✅ 정상 성장
   / 충성도 강화      │      카테고리·상권 모두 작동
                     │
유동 ←───────────────┼───────────────→ 유동
       ↓                                ↑
                     │
   ⛔ 카테고리 위축    │      ⚠️ 비소비형 체류
   상권 자체 기우는    │      유입은 있는데 지갑 안 열림
   중                │      콘텐츠 소비·산책으로 변질
                     │
                    매출 ↓
```

## 사분면별 해석 가이드

### Q1 (유동↑ 매출↑) — ✅ 정상 성장

- **신호**: 카테고리·상권 모두 작동
- **추가 점검**: 임대 시세 함께 오르고 있나? 임대 상승률이 매출 상승률 초과면 BEP 악화 가능
- **다음 액션**: 매출 성장 지속 가능성 검증 (검색 관심도 + 외국인 방문 추세)

### Q2 (유동↑ 매출↓) — ⚠️ 비소비형 체류 (한남 결)

- **신호**: 유입은 있지만 지갑 안 열림. 콘텐츠 소비·산책·SNS 촬영지로 변질
- **추가 점검**:
  - 어떤 카테고리가 가장 빠지고 있나? (의류·잡화·화장품 빠지면 비소비형 체류 강한 신호)
  - 외국인 vs 내국인 구성 변화 (외국인 빠지면 일시적, 내국인도 빠지면 구조적)
  - 검색 관심도 추세 (검색은 유지인데 매출만 빠지면 "보러는 와도 안 사는" 결)
- **다음 액션**: 신규 진입 보류. 본인 매장 차별화 또는 다른 상권 검토.

### Q3 (유동↓ 매출↓) — ⛔ 카테고리 위축

- **신호**: 상권 자체가 기우는 중
- **추가 점검**:
  - 인접 5개 카테고리도 동반 빠지나? (그러면 상권 위축, 본인 카테고리만 빠지면 컨셉 문제)
  - 폐업률·신규개업률 비율 (폐업이 신규 초과면 출구 모드)
- **다음 액션**: 매장 정리 또는 전환 검토. 신규 진입 X.

### Q4 (유동↓ 매출↑) — 🔄 객단가 상승 / 충성도 강화

- **신호**: 적은 사람이 더 쓰는 구조
- **추가 점검**:
  - 객단가 상승률 vs 인플레이션? (인플레이션 초과면 진짜 상승, 미달이면 명목상)
  - 같은 카테고리 다른 매장도 같은 결인가? (그러면 카테고리 트렌드, 우리만 그러면 충성도)
- **다음 액션**: 충성 고객 강화 + 객단가 추가 상승 여지 탐색. 신규 진입은 페르소나 잘 맞는 자리만.

## 추세 판정 규칙

- **추세 판정**: 최근 4분기 평균 vs 이전 4분기 평균 비교
- **±5% 이내 → "유지"** (4분면에서 중심 가까움 표시)
- **±5%~15% → "약한 추세"** (분면 표시 + 약한 신호 라벨)
- **±15% 초과 → "강한 추세"** (분면 표시 + 강한 신호 라벨)
- **데이터 4분기 미만**: 추세 판정 불가, "데이터 부족" 명시

## 한남 사례 결 (Reference Case)

```
한남동 카페 (이미커피)
├── 생활인구: 45,200 → 45,400 (+0.4%, 유지)
├── 매출 카페: 4.2억 → 3.7억 (-12%, 강한 빠짐)
├── 외국인 방문: 1,020 → 520 (-49%, 급감)
├── 검색 "한남동 카페": index 65 → 47 (-28%)
├── 4분면: Q2 비소비형 체류
└── 가설:
    A. 명품·패션 빠짐 + SNS 촬영지로 변질 (높음)
    B. 외국인 → 성수 집중 (중)
    C. 검색 관심 6개월 선행 → 카테고리 약화 (중)
```

## 사용

zone-diagnosis가 Step 2에서 이 도식을 호출하여 4가지 신호 중 하나로 분기. Step 3-5의 카테고리·외국인·검색 데이터로 가설 보강.
