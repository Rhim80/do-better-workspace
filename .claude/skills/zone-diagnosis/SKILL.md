# zone-diagnosis — 기존 상권 시계열 진단

> **역할**: "여기 매출 빠진다, 왜?"에 데이터로 답하는 스킬. 한남동 사례 결.
> **출력**: `data/diagnosis-{zone_name}-{YYYYMMDD}.md` 또는 HTML 한 페이지 진단 카드

---

## 입력

- `personal-context.md` (본인 매장 정보)
- 진단할 zone 이름 (행정동) — 본인 매장이 있는 곳

## 선행 조건

- `data/zone-cache/{zone_name}.json` 존재 (없으면 zone-fetch 자동 호출)

## 프로세스

### Step 1 — 시계열 데이터 추출

zone-cache에서:
- `demand_timeseries.living_pop[]` (분기별 4-8 시점)
- `demand_timeseries.sales_total[]`
- `demand_timeseries.foreign_visit[]`
- `search_interest.monthly[]`

### Step 2 — 4분면 신호 분기 (★ 핵심)

생활인구 추세(↑/↓) × 매출 추세(↑/↓) → 4가지 시그널:

| 유동 | 매출 | 신호 | 해석 |
|------|------|------|------|
| ↑ | ↑ | ✅ 정상 성장 | 카테고리·상권 모두 작동 |
| ↑ | ↓ | ⚠️ **비소비형 체류** | 유입은 있는데 지갑 안 열림. 콘텐츠 소비·산책으로 변질 |
| ↓ | ↓ | ⛔ 카테고리 위축 | 상권 자체가 기우는 중 |
| ↓ | ↑ | 🔄 객단가 상승/충성도 강화 | 적은 사람이 더 쓰는 구조 |

추세 판정: 최근 4분기 vs 이전 4분기 평균 비교. ±5% 이내면 "유지"로 표시.

### Step 3 — 카테고리별 분해

`spending.by_category`에서:
- 본인 카테고리 매출 변화율 (분기 시계열)
- 인접 5개 카테고리 변화율 비교 → "카페만 빠지나, 전반적 위축인가" 판정

### Step 4 — 외국인/내국인 구성 변화

`demand_timeseries.foreign_visit[]` 추세:
- 외국인 회복 중 / 빠짐 / 유지 판정
- "외국인 → 다른 상권 이동" 가설 트리거

### Step 5 — 검색 관심도 시계열

`search_interest.monthly[]`:
- "지역명+업종" 검색 추세 (Google Trends)
- 매출 빠지기 6개월 전 선행 신호로 작동 가능성 점검

### Step 6 — 가설 3개 자동 도출

위 신호 조합으로 가설 후보 생성:
- **가설 A** (예: "비소비형 체류 진행 중") + 근거 인용
- **가설 B** (예: "외국인 회복 더디고 내국인은 선택적 소비") + 근거
- **가설 C** (예: "검색 관심도가 매출보다 먼저 빠졌으니 카테고리 자체 위축 가능성") + 근거

각 가설은 데이터 인용 + 신뢰도(높음/중/낮음).

### Step 7 — 한 페이지 진단 카드 출력

```
┌─────────────────────────────────────────────────┐
│ Diagnosis · 한남동 카페 (이미커피)                │
│ 진단일 2026-05-03                                 │
├─────────────────────────────────────────────────┤
│ 4분면 신호: ⚠️ 비소비형 체류                      │
│   생활인구  +3% (유지)                           │
│   매출      -12% (빠짐)                          │
├─────────────────────────────────────────────────┤
│ 카테고리 변화율 (최근 4Q vs 이전 4Q)              │
│   카페      -12% ⬇                              │
│   한식       -5%                                 │
│   양식       +2%                                 │
│   의류·잡화 -18% ⬇⬇                             │
│   화장품    -22% ⬇⬇⬇                           │
├─────────────────────────────────────────────────┤
│ 외국인 방문: -36% (회복 약함)                     │
│ 검색 관심도: "한남동 카페" -28% (선행 빠짐)       │
├─────────────────────────────────────────────────┤
│ 가설 3개:                                        │
│   A. 비소비형 체류 + 명품·패션 빠짐 (신뢰 높음)   │
│   B. 외국인 → 성수 집중 (신뢰 중)                │
│   C. 검색 관심 6개월 선행 → 카테고리 약화 (중)    │
├─────────────────────────────────────────────────┤
│ 다음 액션: entry-verdict로 다른 상권 점프 검토    │
└─────────────────────────────────────────────────┘
```

## 출력 형식

`references/diagnosis-template.md` 사용. zone-report로 HTML/PDF 변환 가능.

## 체크리스트

- zone-cache에 시계열 데이터(quarters 4개+) 있는가?
- 본인 카테고리가 `personal-context.md`에 명시됐는가?
- Google Trends 데이터 ok=true인가? (선행 신호 분석용)

## 주의

- 4분기 미만이면 추세 판정 불가 → "데이터 부족" 명시
- 외국인 방문은 행정동 단위 정밀도 한계 → "관광특구 단위 추정" 라벨
- 가설은 데이터 패턴일 뿐 인과 X. "현장 답사·고객 인터뷰로 검증" 명시 필수
