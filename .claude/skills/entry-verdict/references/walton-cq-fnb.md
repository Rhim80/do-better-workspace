# Walton CQ + Toulmin (외식업 출점 의사결정 특화)

> entry-verdict가 호출하는 논증 프레임워크. F&B 도메인 특화.

## Toulmin 6요소 (외식업 신규 출점)

| 요소 | 외식업 의미 | 데이터 출처 |
|------|------------|-------------|
| **Claim** | "{zone}는 {brand}에 진입해야 한다 / 보류해야 한다 / 들어가지 말아야 한다" | entry-verdict Verdict |
| **Grounds** | 5-Layer 점수 + Magnitude 결과 + 7 CQ 점검 결과 | zone-cache |
| **Warrant** | "Volume·Rhythm·Spending이 P50+ 이고 폐업률 P75 미만이면 외식업 진입 가능하다" — 조건부 일반화 | F&B 산업 통계 (외식업체 1년 생존율, 폐업률) |
| **Backing** | 외식업 신규점 1년 생존율 70% (한국외식산업연구원 2023). 5년 생존율 ≈40%. 폐업률 P75 초과 상권에서 신규점 1년 폐업률 2.3배 | references/magnitude-estimation.yaml |
| **Qualifier** | "낙관 시나리오 P75 평당매출 달성 시", "임대 협상 가능 시" 등 — 어떤 조건에서만 유효 | Magnitude·임대 시세 비교 결과 |
| **Rebuttal** | "임대 시세가 BEP 25% 초과 시 무효", "본인 매장 5km 이내 카니발 시 무효" 등 | 7 CQ Rebuttal 트리거 조건 |

## 7 Critical Questions (외식업 특화)

각 CQ에 충족(✅) / 미충족(❌) / 판단보류(⚠️) + 근거 1-2문장.

### CQ1. Volume — 페르소나 살고/머무는가?

- **임계**: 거주 + 생활인구 percentile 평균 ≥ P50 (서울 기준)
- **충족 예**: "P72 — 거주 P55 + 평일 평균 생활인구 P88, 페르소나(20-40 직장인) 비중 P75"
- **미충족 예**: "P32 — 거주 P22, 야간 생활인구는 높으나 운영시간대 매칭 P28"
- **보류 예**: "거주 데이터 결측, 생활인구만 P65"

### CQ2. Rhythm — 운영시간대에 페르소나 있는가?

- **임계**: 본인 운영시간(예: 11:00~22:00) ↔ 시간대별 생활인구 매칭 비율 ≥ 40%
- **충족**: "런치+디너 시간대 생활인구 합산이 일평균의 52%"
- **미충족**: "야간(21-04) 운영인데 해당 시간대 생활인구 일평균의 18%"

### CQ3. Spending — 업종에 실제 돈 쓰는가?

- **임계**: 동 카테고리 추정매출 평당 ≥ 본인 카테고리 서울 P50
- **충족**: "한식 P67, 평당 분기매출 1,250만"
- **미충족**: "카페 P34, 평당 분기매출 480만 — 객단가 낮은 양산형 위주 가능성"
- **주의**: 객단가 낮은 곳일수록 회전율로 보완 가능. CQ6과 함께 봐야 함.

### CQ4. Saturation — 카테고리 작동? 틈 있나?

- **임계**: 폐업률 P75 미만 + 동일 컨셉 점포 < 5개
- **충족**: "한식·구이 4개 점포 (P31), 폐업률 P52"
- **미충족**: "카페 92개 (P98), 폐업률 P78 — 진입 시 차별화 필수"
- **보류**: "동일 컨셉 8개지만 평균 객단가가 본인 컨셉과 1.8배 차이 → 컨셉 적합 별도 검토"
- **주의**: 폐업률 단독 판정 금지. 신규개업률·점포수 절대값과 함께.

### CQ5. Cannibal — 본인 기존 매장 영향 없나?

- **임계**: 거리 ≥ 3km 또는 영업시간 분리
- **충족**: "본점 4.2km, 동일 시간대 운영, 페르소나 분리(본점 직장인 / 후보 거주민)"
- **미충족**: "본점 1.8km, 동일 컨셉·페르소나 → 매출 갉아먹기 30~50% 위험"
- **보류**: "거리는 5km이나 같은 페르소나(20대 직장인) → SNS 노출 중복으로 신규 효과 제한 가능"

### CQ6. Concept — 객단가·컨셉 일관성?

- **임계**: 본인 객단가가 상권 동 카테고리 P30~P75 범위 내 (너무 비싸도 너무 싸도 부적합)
- **충족**: "이미커피 객단가 9,500원 = 한남 카페 P58, 자연스러운 위치"
- **미충족 (싸도 미충족)**: "객단가 6,500원 = 한남 카페 P12 — 상권 평균 1.4만에 비해 너무 저가, 손님이 못 찾거나 충성도 약함"
- **미충족 (비싸도 미충족)**: "객단가 18,000원 = 강남 P92 — 너무 프리미엄, 회전율 부족 위험"

### CQ7. Magnitude — 임대 정당화?

- **임계**: Δ% = (시세 - BEP) / BEP
  - Δ% ≤ 10% → 충족
  - 10% < Δ% ≤ 30% → 보류 (협상 여지)
  - Δ% > 30% → 미충족
- **충족**: "BEP 1,200만 시세 1,250만 Δ +4%"
- **보류**: "BEP 550만 시세 720만 Δ +31% — 600만 협상 시 수치 충족"
- **미충족**: "BEP 480만 시세 750만 Δ +56% — 임대 자체가 손익분기 깨뜨림"

## Walton 도식 (논증 평가)

```
[Grounds: 데이터 인용] ──Warrant──> [Claim: Verdict]
                          │
                          ↓
                     [Backing: 산업 통계]
                          │
                          ↓
              [Qualifier: 조건부 유효]
                          │
                          ↓
                  [Rebuttal: 무효 조건]
```

각 CQ는 Grounds-Warrant-Backing 흐름으로 검증되며, Rebuttal 조건이 활성되면 Verdict 자체를 뒤집음.

## 사용

`entry-verdict` Step 3·4에서 이 파일을 직접 임포트하여 Toulmin 6요소 채움 + 7 CQ 점검. 출력은 `references/entry-verdict-template.md` 카드 형식으로 합쳐짐.
