# Commands

사용 가능한 Slash Commands 목록입니다.

## 초기 설정

- `/setup-workspace` - 워크스페이스 초기 설정 마법사

## Daily Workflow

- `/daily-note` - 오늘 Daily Note 생성/열기
- `/daily-review` - 전날/오늘 변경사항 분석

## 지식 관리

- `/idea [카테고리]` - 대화에서 아이디어 추출 및 저장
- `/todo` - 빠른 할 일 캡처
- `/todos` - 할 일 목록 보기 및 관리

## AI 활용

- `/thinking-partner` - 소크라테스식 질문으로 생각 발전

## 시스템

- `/create-command` - 커스텀 명령어 생성

---

각 명령어의 자세한 사용법은 해당 파일을 참고하세요.
