## 상황 설정

**월요일 오전 10시. 정동 오아시스 덕수궁.**

창밖으로 덕수궁 돌담길이 보인다. 3월의 서울은 아직 쌀쌀하지만, 벚꽃이 슬슬 기지개를 켜고 있다.

당신은 HFK(HBR Forum Korea)에 합류한 지 딱 1주일 된 신입이다. 운영 파트너 슬기님 아래에서 일하고 있다.

슬기님이 아메리카노 두 잔을 들고 자리에 앉는다. 하나를 건네며 말한다.

> **슬기님**: 커피. 자판기 커피긴 한데 여기 자판기 커피가 은근 괜찮아.
>
> 그건 그렇고, 오늘 재윤님이랑 봄 시즌 리뷰 미팅이 있어.
> 아시겠지만 이번 봄 시즌이 좀... 그래.
>
> 작년 가을에 203명까지 갔거든? 역대 최고.
> 근데 이번 봄이 152명이야.
> 재윤님이 수치 보고 이틀 동안 말이 없으셨어.
>
> 그러다가 갑자기 슬랙에 "괜찮다 우리에겐 여름이 있다" 올리시고
> 바로 다음 메시지가 "여름 시즌 200명 복귀 전략 가져와"
>
> ...그게 오늘 미팅이야.

**(속으로)**: 합류 1주일 만에 200명 복귀 전략이라니. 일단 데이터부터 보자.

슬기님이 노트북을 돌려서 보여준다.

> **슬기님**: 여기 데이터 4개 있어.
>
> - 이번 봄 시즌 멤버 등록 데이터
> - 지난 겨울 시즌 피드백
> - 지난 겨울 이벤트 참석 데이터
> - 시즌별 수익 요약
>
> 참고로 재윤님이 저번에 Claude한테
> 슬랙 9년치 던져서 "우리가 말만 하고 안 한 거 뭐야?" 물어봤거든?
> Claude가 줄줄이 읊었어. 뉴스레터 4회, 팟캐스트 3회, 유튜브 7회...
> 재윤님 표정이 점점 굳어지는 거 봤으면 ㅋㅋ
>
> 그래서 이번엔 "데이터 기반으로 하자"가 모토야.
> Claude Code 써서 분석해봐. 나도 배우고 싶어.
>
> 아 그리고 54만원 커피 자판기 이슈는... 피드백에서 또 나올 거야.
> 그건 나중에 생각하자.

**(속으로)**: 54만원 멤버십에 자판기 커피... 이건 진짜 개선점인 건가 밈인 건가.

- 진짜 마감: 오늘 오후 3시 미팅
- 체감 마감: 점심 전 (슬기님 검토 시간 필요)
- 실제 가용 시간: 지금부터 약 3시간
- 받은 파일: 멤버 등록, 피드백, 이벤트, 수익 데이터

**Claude Code가 없었다면?**
- 152명 멤버 데이터 엑셀 피벗 (1시간)
- 165개 피드백 텍스트 일일이 읽기 (2시간)
- 30개 이벤트 참석률 계산 (30분)
- 시즌별 트렌드 차트 만들기 (1시간)
- 인사이트 정리 및 전략 초안 (2시간)
- 총 **6시간 30분, 오후 미팅까지 절대 불가능**

**Claude Code가 있다면?**
- 대화로 분석 요청 (30분)
- 결과 확인하고 추가 질문 (30분)
- 전략 정리 (30분)
- 총 **1시간 30분, 점심 전에 끝남**

자, 시작하자. 슬기님이 지켜보고 있다.

## 미션 1: 봄 시즌 멤버 분석 - 152명의 진실

슬기님이 팔짱을 끼고 말한다.

> **슬기님**: 일단 152명이 누군지부터 파악하자.
> 신규가 몇 명이고, 재등록이 몇 명인지.
> 어떤 팀에 몰렸는지도 중요해.
>
> 아, 그리고 AI부사수 팀 있잖아.
> 1지망 경쟁률이 장난 아니었거든.
> 그거 정확히 몇 대 몇인지 좀 뽑아줘.
>
> 재윤님이 "AI팀은 왜 맨날 먼저 마감이야?" 하실 거야. 근거 필요해.

**(속으로)**: 먼저 전체 그림부터 보고, 팀별로 쪼개보자.

**분석 요청**:
```
"@hfk_members_2026spring.csv 분석해줘.
신규 vs 재등록 비율, 팀별 배정 인원,
1지망 팀 vs 실제 배정 팀 매칭률 봐줘.
특히 AI부사수 팀 1지망 경쟁률이 궁금해.
가입경로별 분포도 정리해줘 - 어디서 유입이 많은지"
```

**결과 확인 후 추가 질문 예시**:
```
"재등록 멤버 중에 3회 이상 연속 등록한 충성 멤버는 몇 명이야?
이 사람들의 팀 선호도가 전체 평균이랑 다른지 봐줘"
```

## 미션 2: 피드백 해부 - 54만원의 가치

미션 1 결과를 정리하고 있는데 슬기님이 의자를 끌고 옆에 앉는다.

> **슬기님**: 오 벌써? 빠르다.
>
> 자 이제 무서운 거 볼 차례야. 피드백.
> 지난 겨울 시즌 끝나고 받은 거거든.
>
> 솔직히 내가 다 못 읽었어. 165개잖아.
> 그냥 만족도 숫자만 봤는데...
> 가성비 점수가 좀 아프더라.
>
> 아 그리고 누가 "슬기님 오타 줄었어요 감동" 이라고 썼다며?
> ...그건 좋은 건지 나쁜 건지 모르겠어.
>
> 텍스트 피드백에서 패턴 좀 뽑아줘.
> 재윤님이 "데이터로 말해"를 요즘 입에 달고 사니까.

**(속으로)**: 165개 피드백 텍스트를 Claude가 읽으면... 꽤 재밌는 게 나올 것 같은데.

**분석 요청**:
```
"@hfk_feedback_2025winter.csv 분석해줘.
만족도 항목별 평균 비교하고, 팀별로 차이가 있는지 봐줘.
가성비만족도가 낮다고 들었는데 어느 정도인지.
좋았던점/아쉬운점 텍스트에서 자주 나오는 키워드 패턴도 뽑아줘.
NPS 점수 분포도 보고 싶어"
```

**결과 확인 후 추가 질문 예시**:
```
"가성비 점수가 낮은 사람들의 공통점이 뭐야?
신규 멤버가 더 낮은지, 재등록 멤버가 더 낮은지?
재등록 의향이 '낮음'인 사람들의 아쉬운점에서 패턴 뽑아줘"
```

## 미션 3: 이벤트 성적표 - 을지로 노가리의 비밀

슬기님이 커피를 한 모금 마시고 (자판기 커피다) 말한다.

> **슬기님**: 이벤트 데이터도 봐야 해.
> 겨울 시즌에 이벤트를 꽤 했거든.
>
> 도쿄 필드트립은 신청이 폭주했어.
> 정원 20명인데 60명 넘게 신청했나? 그래서 추첨했거든.
>
> 근데 웃긴 게... 제일 만족도 높은 이벤트가
> 공식 이벤트가 아니라 번개모임이야.
> 을지로 노가리 ㅋㅋ
>
> 재윤님이 이거 보면 "우리가 뭘 잘못하고 있는 거 아냐?" 할 거야.
>
> 노쇼 패턴도 봐줘. 차박은 노쇼 0이었는데
> 다른 건 좀 있었거든.

**(속으로)**: 을지로 노가리가 도쿄 필드트립보다 만족도가 높다... 이건 인사이트가 될 수 있겠는데.

**분석 요청**:
```
"@hfk_events_2025winter.csv 분석해줘.
이벤트별 신청률(신청/정원), 참석률(참석/신청), 노쇼율 비교하고,
유형별(정기/특별/번개) 만족도 차이를 봐줘.
만족도 TOP 3, 신청 경쟁률 TOP 3 뽑아주고,
노쇼가 많은 이벤트 패턴도 분석해줘"
```

**결과 확인 후 추가 질문 예시**:
```
"번개모임 만족도가 높은 이유가 뭘까?
공식 이벤트와 뭐가 다른 건지 가설을 세워줘.
여름 시즌 이벤트 기획에 반영할 수 있는 인사이트는?"
```

## 미션 4: 수익 트렌드 - 하락의 원인

벌써 11시 반. 슬기님이 점심 메뉴를 고민하다가 갑자기 진지해진다.

> **슬기님**: 마지막으로 이게 제일 중요해.
> 시즌별 수익 추이.
>
> 작년 가을에 203명으로 피크 찍고
> 겨울 168명, 이번 봄 152명.
> 2시즌 연속 하락이야.
>
> 재윤님이 제일 신경 쓰는 건
> "왜 떨어졌는지"랑 "여름에 어떻게 200명 돌아가는지"야.
>
> 등록수만 보지 말고 재등록률 추이,
> 이벤트비 대비 효과, 영업이익 흐름까지 봐줘.
>
> 아 그리고... 해외 필드트립 있는 시즌이
> 이벤트비가 확 올라가거든.
> 그게 이익에 얼마나 영향 주는지도.

**(속으로)**: 203명 → 168명 → 152명. 트렌드가 좋지 않다. 근데 원인이 뭘까?

**분석 요청**:
```
"@hfk_revenue_seasons.csv 분석해줘.
시즌별 등록수, 재등록률, 영업이익 추이를 보여줘.
비용 구조(공간/이벤트/운영)가 시즌마다 어떻게 다른지,
해외 필드트립 유무에 따른 이익률 차이도 분석해줘.
2시즌 연속 하락의 원인 가설을 데이터 기반으로 세워줘"
```

**결과 확인 후 추가 질문 예시**:
```
"여름 시즌에 200명 달성하려면 신규 유입이 얼마나 필요해?
재등록률을 현재 수준으로 유지한다고 가정하고 시뮬레이션해줘"
```

## 미션 5: 재윤님 보고용 여름 시즌 전략

시계를 본다. **12시 10분.** 슬기님이 김밥을 사 들고 돌아온다.

> **슬기님**: 먹으면서 해. 자 마지막이야.
>
> 지금까지 분석한 거 다 합쳐서
> 재윤님 보고용 1장 요약 만들어줘.
>
> 재윤님 스타일 알지?
> 슬랙 9년치를 Claude한테 던지는 분이야.
> 숫자 없으면 안 넘어가.
>
> 포맷은 이렇게:
> - 봄 시즌 핵심 팩트 3개 (현실 직시)
> - 하락 원인 진단 3개 (왜 떨어졌나)
> - 여름 시즌 200명 달성 액션 아이템 3개 (어떻게 돌아가나)
>
> 차트도 넣어줘. 재윤님이 시각적인 거 좋아하셔.
> 컨설턴트 출신이라 숫자만 나열하면 "So What?" 하셔.

**(속으로)**: 컨설턴트 출신에게 보고하는 건 처음인데... 데이터가 있으니 괜찮겠지.

**분석 요청**:
```
"지금까지 분석한 4개 파일 전부 종합해서 재윤님 보고용 1페이지 요약 만들어줘.
포맷:
- 봄 시즌 핵심 팩트 3개 (152명, 재등록률, 신규 유입 경로 등)
- 하락 원인 진단 3개 (데이터 기반 근거 포함)
- 여름 시즌 200명 달성 전략 3개 (구체적 액션 아이템)
각 항목에 숫자 근거 필수.
시즌별 등록 추이 차트도 넣어줘"
```

**결과 확인 후 추가 질문 예시**:
```
"이 전략의 예상 비용을 산출해줘.
200명 달성 시 예상 매출이랑 비용 시뮬레이션도"
```

## 에필로그

**오후 2시 50분.**

슬기님에게 분석 결과를 보여준다.

> **슬기님**: ...이걸 오전에 다 한 거야?
>
> 잠깐. 을지로 노가리가 도쿄 필드트립보다 만족도 높다는 거...
> 이거 재윤님 앞에서 말해도 돼?
>
> 아니 말해야 해. 데이터가 말하는 거니까.
>
> 근데 "54만원인데 커피는 왜 자판기" 피드백은 빼자.
> ...아니 넣자. 솔직한 게 좋지.

3시. 재윤님이 들어온다.

> **재윤님**: 시작하죠. 뭐가 나왔어요?

당신이 화면을 공유한다. 시즌별 등록 추이 차트가 뜬다.
203 → 168 → 152. 하락 곡선이 선명하다.

> **재윤님**: ...
>
> (5초 침묵)
>
> 원인이 뭐래요?

데이터 기반 진단 3가지를 설명한다. 재등록률 하락, 신규 유입 채널 편중, 가성비 인식 문제.

> **재윤님**: 가성비? 54만원이 비싸다는 거야?
>
> **슬기님**: 아뇨, 가격이 아니라 체감 가치예요. 피드백 보시면...
>
> **재윤님**: (피드백 텍스트를 읽으며) "커피는 왜 자판기"...
> 이건 진짜 고쳐야 하나?
>
> **슬기님**: 재윤님, 그건 밈이에요. 걱정하실 필요 없...
>
> **재윤님**: 아니, 커피 머신 하나 놓죠. 그거 한 대에 200만원이야.
> 멤버 4명이 이것 때문에 재등록하면 본전이잖아.

**(속으로)**: 컨설턴트 출신의 ROI 계산 속도란...

> **재윤님**: 여름 전략은?

200명 달성 액션 아이템 3가지를 설명한다.

> **재윤님**: (고개를 끄덕이며) 괜찮네요.
> 근데 이거 얼마나 걸렸어요?
>
> **당신**: 오전에 다 했습니다.
>
> **재윤님**: ...Claude Code?
>
> **당신**: 네.
>
