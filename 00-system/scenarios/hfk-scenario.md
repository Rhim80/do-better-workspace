# 데이터 분석 실습 시나리오 - HFK (HBR Forum Korea)

> **선행 조건**: `claude-code-practice-guide.md`의 Part 1-2 완료
> **샘플 데이터**: `50-resources/sample-data/hfk/`

---

## 상황

당신은 HFK에 합류한 지 1주일 된 신입이다. 운영 파트너 슬기님 아래에서 일하고 있다.

HFK는 직장인 리더십 커뮤니티다. 3개월 시즌제로 운영되고, 멤버십은 54만원. 작년 가을에 203명으로 역대 최고를 찍었는데, 이번 봄 시즌 등록이 152명에 그쳤다. 2시즌 연속 하락. 재윤님(대표)이 수치 보고 이틀 동안 말이 없다가, 슬랙에 딱 한 줄 올렸다.

> "괜찮다. 우리에겐 여름이 있다."

바로 다음 메시지: **"여름 시즌 200명 복귀 전략 미팅. 오늘 오후 3시."**

슬기님이 데이터 4개를 슬랙으로 보내며 말한다.

> 오전 안에 분석해서 정리해봐.
> 재윤님은 삼성SDS 12년 하신 분이야. 숫자 없으면 안 넘어가.
> "So What?"이 입버릇이거든.
>
> Claude Code 써. 파일 4개 한번에 던지면 돼. 내가 쓸 때는 이게 제일 빨라.
> 저번에 재윤님이 슬랙 9년치 던져서 "우리가 말만 하고 안 한 거 뭐야?" 물어봤는데
> Claude가 뉴스레터 4회, 팟캐스트 3회, 유튜브 7회... 미실행 항목을 줄줄이 읊었어.
> 회의실이 5초간 조용해졌거든. 그 5초가 꽤 길었어.
> 이번엔 우리가 먼저 데이터로 준비하자.
>
> 아 그리고 멤버 데이터에 "강동원"이라는 사람 있는데 동명이인이야. 놀라지 마.

### 데이터

| 파일 | 내용 | 행 |

## 분석 요청

```
"50-resources/sample-data/hfk/ 안에 있는 4개 파일을 전부 분석해줘.

배경: HFK는 직장인 리더십 커뮤니티야. 시즌제 54만원 멤버십.
작년 가을 203명(역대 최고) → 겨울 168명 → 이번 봄 152명. 2시즌 연속 하락 중.
여름 시즌에 200명 복귀가 목표야.

알고 싶은 것:
1. 왜 떨어졌는지 - 재등록률, 신규 유입, 피드백, 수익 데이터를 교차해서 원인 진단
2. 어디에 기회가 있는지 - 팀별 인기도, 이벤트 만족도, 가입 경로 효율, 텍스트 피드백 패턴
3. 여름 시즌 200명 달성하려면 뭘 해야 하는지 - 구체적 액션 아이템

@community-health-framework.md 랑 @season-growth-framework.md 관점으로 분석해줘.
차트도 만들어줘."
```

### 후속 질문 예시

분석 결과가 나오면 슬기님이 옆에서 볼 거다. 이런 질문을 던져보자:

```
"이걸 재윤님 보고용 1페이지로 정리해줘.
핵심 팩트 3개, 원인 진단 3개, 액션 아이템 3개.
각각 숫자 근거 포함해서. 재윤님이 'So What?' 하면 바로 답할 수 있게."
```

## 더 파고들 수 있는 질문

보고서 초안이 나오면 슬기님이 "하나만 더" 할 거다. 대비해두자:

```
"재등록률을 5%p 올리는 것과 신규 50명 더 모으는 것 중 뭐가 현실적이야? 비용 시뮬레이션해줘"
```

```
"피드백에서 가성비 불만이 나오는데, 가격을 안 건드리고 체감 가치를 올릴 방법은?"
```

```
"번개모임 만족도가 공식 이벤트보다 높은 이유가 뭘까? 여름 시즌 이벤트 기획에 어떻게 반영할 수 있어?"
```

> 팁: 분석이 끝나고 시간이 남으면 "8시즌 연속 재등록한 멤버가 있는데, 이 사람이 왜 남았는지 역추적해줘"를 던져봐. 충성 멤버의 패턴이 여름 전략의 힌트가 될 수 있다.
