## 상황 설정

**월요일 오전 10시. 가산동 사무실.**

4월 합정 이사를 앞두고 분주한 사무실. 3월 실적을 정리해야 하는데, 데이터가 여기저기 흩어져 있다. 자사몰은 Cafe24 어드민, 무신사는 파트너센터, 올리브영은 입점 관리 시스템, 카카오선물하기는 또 다른 대시보드...

5개 채널의 데이터를 하나하나 열어 비교하고 있으면 하루가 다 간다.

슬랙에 메시지가 올라온다.

> **상찬님**: 3월 전채널 매출 정리 필요해요
> 화이트데이 때 선물세트 밀었는데 실제로 효과 있었는지 보고 싶고
>
> 그리고 채널별로 수수료가 다 다른데
> 매출만 보면 안 되잖아요
> 실제로 우리 손에 남는 게 얼마인지 알아야 해요
>
> 무신사가 매출은 높은데 수수료 25%니까
> 실 마진으로 보면 자사몰이 나을 수도 있거든요
>
> 아, 그리고 경쟁사 가격 비교도 해주세요
> 탬버린즈, 논픽션이랑 우리 포지셔닝이 맞는 건지
> 인스타 콘텐츠가 실제 매출에 어떤 영향을 주는지도 궁금해요

다른 슬랙도 온다.

> **디자이너**: 패키지 디자인 리뉴얼 회의가 수요일인데
> 채널별로 잘 나가는 상품이 다를 것 같거든요
> 올리브영 고객이랑 29CM 고객이 같은 걸 살까요?
>
> **마케터**: 인스타 3월 성과 보고서도 필요해요
> 릴스 vs 피드 vs 캐러셀 중에 뭐가 제일 효과 좋은지
> 화이트데이 프로모션 콘텐츠가 실제로 먹혔는지...

**(속으로)**: 매출 데이터 5개 채널, 상품 마스터, 인스타그램 성과, 경쟁사 가격... 파일 4개를 교차 분석해야 한다. 엑셀로 하면 이틀은 걸리겠는데.

- 진짜 마감: 수요일 오전 디자인 회의
- 받은 데이터: 3월 전채널 매출(2,137건), 상품 마스터(30개), 인스타그램(55건), 경쟁사(31개)
- 분석 관점: 채널별 수익성, 상품 포트폴리오, 콘텐츠 ROI, 경쟁 포지셔닝

## Mission 1: 전채널 매출 분석 + 수익성 대시보드

상찬님이 가장 알고 싶은 것: **"매출 말고, 실제로 남는 돈이 얼마야?"**

5개 채널의 수수료율이 전부 다르다. 무신사 25%, 올리브영 30%, 카카오선물하기 28%, 29CM 22%, 자사몰 3.5%(PG수수료만). 매출 1위 채널이 수익 1위가 아닐 수 있다.

거기에 화이트데이(3/14) 시즌 효과까지 봐야 한다. 선물세트를 카카오선물하기에 집중 밀었는데, 수수료 28%를 감안해도 의미 있었는지.

> **상찬님**: 단순히 채널별 매출 순위만 보면 안 돼요
> 수수료 떼고, 원가 빼고, 진짜 남는 마진을 봐야 해요
> 채널별로 어떤 카테고리가 수익성이 좋은지도요
>
> 화이트데이 전후로 끊어서 비교해주세요
> 기획전 효과가 있었는지 숫자로 봐야 합니다

### 분석 요청

```
"@liber_channel_sales_2603.csv 랑 @liber_products_master.csv 두 파일 교차 분석해줘.

1. 채널별 매출 현황
   - 5개 채널(자사몰, 무신사, 올리브영, 카카오선물하기, 29CM) 매출 비교
   - 주문 건수, 평균 객단가, 총 매출

2. 채널별 수익성 분석 (핵심)
   - 매출에서 수수료 차감한 실 수익 계산
   - 원가까지 반영한 순이익률 비교
   - 채널별 x 카테고리별 마진 매트릭스

3. 화이트데이 효과 분석
   - 3/10~14 vs 일반 기간 매출 비교
   - 선물세트 카테고리 매출 추이 (일별)
   - 카카오선물하기 채널의 화이트데이 전후 변화

4. 채널별 베스트셀러 TOP 5
   - 채널마다 잘 나가는 상품이 다른지 확인"
```

### 결과 확인 후 후속 질문

```
"매출 1위 채널이랑 순이익 1위 채널이 다르면 어떤 전략이 필요해?
자사몰 비중을 늘리려면 어떻게 해야 할지 제안해줘"
```

```
"화이트데이 기획전 ROI를 계산해줘.
선물세트 추가 매출 vs 카카오선물하기 수수료 비용, 순이익 기준으로"
```

```
"이 분석 결과를 상찬님 보고용 1페이지 요약으로 만들어줘.
핵심 숫자 3개, 인사이트 3개, 다음 달 액션 아이템 3개 포맷으로"
```

### 도전 과제

```
"채널별 매출 추이 라인차트, 카테고리별 마진 히트맵, 화이트데이 전후 비교 바차트를
HTML 대시보드 하나로 만들어줘"
```

## Mission 2: 경쟁사 포지셔닝 + 인스타그램 콘텐츠 ROI

마케터가 매주 고민하는 것: **"우리 콘텐츠가 실제로 매출에 도움이 되고 있나?"**

인스타그램에 열심히 올리고 있지만, 좋아요 수가 곧 매출은 아니다. 릴스가 도달은 높은데 실제 구매로 이어지는 건 캐러셀일 수 있다. 거기에 경쟁사(탬버린즈, 논픽션, 이솝)와 가격/포지셔닝 비교까지 하면 전체 그림이 보인다.

> **마케터**: 릴스에 시간 많이 쓰는데 진짜 효과 있는 건지 모르겠어요
> 도달은 높은데 저장이나 웹사이트 클릭은 캐러셀이 나은 것 같기도 하고
>
> 경쟁사 대비 우리 가격이 어디쯤인지도 정리해주세요
> 탬버린즈 핸드크림이 38,000원인데 우리는 16,000원이잖아요
> 이 차이를 어떻게 마케팅에 활용할 수 있을지...

### 분석 요청

```
"@liber_instagram_2603.csv, @liber_competitor_pricing.csv, @liber_channel_sales_2603.csv
세 파일을 교차 분석해줘.

1. 인스타그램 콘텐츠 성과 분석
   - 콘텐츠 유형별(릴스/피드/캐러셀/스토리) 평균 도달, 좋아요, 저장, 웹사이트클릭 비교
   - 주제 분류별(제품/교육/UGC/브랜딩/프로모션/콜라보) 성과 차이
   - 가장 웹사이트 클릭이 높았던 콘텐츠 TOP 5

2. 콘텐츠 → 매출 연결 분석
   - 인스타 게시물 올린 날 vs 안 올린 날의 자사몰 매출 차이
   - 프로모션 콘텐츠 게시일 전후 3일간 매출 변화
   - 상품태그된 상품의 해당 주 매출 vs 태그 안 된 상품 매출

3. 경쟁사 가격 포지셔닝
   - 카테고리별 가격 비교 (리베르 vs 탬버린즈 vs 논픽션 vs 이솝 vs 그라넬 vs 딥티크)
   - 용량 대비 가격(ml당 단가) 비교
   - 리뷰 수 대비 평점 스캐터

4. 포지셔닝 갭 분석
   - 리베르가 경쟁사 대비 강한 카테고리 vs 약한 카테고리
   - 가격 갭이 큰 카테고리에서의 기회 (예: 바디워시는 논픽션만 있음)"
```

### 결과 확인 후 후속 질문

```
"릴스 도달 대비 실제 전환율이 가장 높은 콘텐츠 유형이 뭐야?
다음 달 콘텐츠 캘린더를 짠다면 어떤 유형에 비중을 둬야 해?"
```

```
"경쟁사 대비 우리 강점을 한 문장으로 정리해줘.
'탬버린즈 핸드크림의 절반 가격에, 무신사 리뷰 1,695개의 검증된 품질' 같은 느낌으로"
```

```
"이 분석을 마케터가 바로 쓸 수 있는 경쟁사 비교 대시보드로 만들어줘.
카테고리별 가격 포지셔닝 차트 + 콘텐츠 유형별 ROI 비교 포함해서"
```

### 도전 과제

```
"경쟁사 비교 차트(가격 x 리뷰수 스캐터, 카테고리별 가격 바차트)와
인스타 성과 차트(유형별 도달 vs 전환, 주제별 engagement rate)를
하나의 HTML 리포트로 만들어줘"
```

### 데이터 간 연결고리

```
[상품 마스터] --SKU-- [채널 매출]
     |                    |
  원가/수수료          주문/매출
     |                    |
  순이익 계산         채널별 성과
                         |
[인스타그램] --상품태그/날짜-- [채널 매출]
     |
  콘텐츠 성과 → 매출 연결

     |
  포지셔닝 비교
```
