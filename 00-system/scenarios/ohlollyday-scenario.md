> **샘플 데이터**: `50-resources/sample-data/ohlollyday/`

## 상황 설정

**월요일 아침 10시. 성수동 오롤리데이 사무실.**

조이는 모니터 앞에서 브라우저 탭을 세고 있다. 카페24 어드민, 네이버 스마트스토어 센터, 29CM 파트너 어드민, 카카오선물하기 파트너센터, 텐바이텐 셀러 페이지, 핫트랙스 정산 메일. 6개 탭. 매주 월요일마다 반복되는 일이다.

통계 페이지가 있는 채널은 그나마 낫다. 없는 채널은 주문 데이터를 직접 추출해서 주문번호 기준으로 중복을 제거하고, 취소/환불 건을 빼고, 실질 주문건수를 산출한다. 제품 판매 데이터가 없는 채널은 피봇팅해서 판매금액/판매수량 기준으로 제품 순위를 직접 계산해야 한다. 6개 채널에서 똑같은 패턴을 6번 반복하면 2시간이 지나 있다.

그런데 오늘은 좀 다르다. 롤리 대표가 금요일에 슬랙에 남긴 메시지 때문이다.

> **롤리**: 수요일에 Q2 채널 전략 정하자. 카카오선물하기 수수료가 30%인데 우리가 거기서 진짜 남는 게 있는 건지 한번 보고 싶어. 그리고 인스타 광고 2월부터 늘렸는데 자사몰 유입이 실제로 늘었는지도.

조이는 안다. 매주 화요일 오전 주간 피드백 회의 때 이 데이터가 필요하다는 것도 안다. 하지만 6개 채널의 숫자를 모아서 "남는 돈"까지 계산한 적은 없다. 매출은 봤지만 채널 수수료를 빼고 원가를 빼서 채널별 영업이익을 비교한 적은 없다.

옆자리에서 썸머가 한숨을 쉬고 있다. 썸머도 매주 월요일이면 같은 고통이다. 인스타그램 계정 인사이트, 게시물별 개별 인사이트를 확인하고, 링크트리 클릭수를 보고, GA에서 채널별 트래픽과 매출을 뽑는다. 그걸 구글시트에 넣고, 같은 데이터를 노션 주간회의 페이지에 또 넣는다. 이중 입력. GA 데이터는 네이버 모바일/PC 같은 UTM별로 항목이 나뉘어 있어서 직접 합산해야 하는데, 이 계산에서 실수가 나기 쉽다. 전주 비교를 하려면 지난주 노션 페이지를 계속 띄워놔야 한다. 이것만 2~3시간이다.

**(조이, 속으로)**: 수요일까지 3일. 매출이 아니라 '남는 돈' 기준으로 채널별 수익성을 보여줘야 한다.

- 마감: 수요일 오전 10시 (롤리 대표 + 전체 팀미팅)
- 파일: 주문 1,920건, 상품 35개, 인스타 콘텐츠 50건

## 미션 1: 채널별 진짜 수익, 얼마나 남아?

> **뵤**: 조이씨, 나 3월 정산하면서 봤는데 카카오선물하기 수수료가 생각보다 많이 빠지더라. 정산서 가공해서 이카운트에 넣을 때마다 느끼는데, 채널별로 실제 마진이 얼마인지 한눈에 보이는 게 없어.
>
> **이정원**: 핫트랙스도 그래요. 포털시스템에서 지점별 매출 보고 구글 스프레드시트에 수기로 옮기는데, 수수료 25% 빼면 남는 게 있는 건지 모르겠어요.

**분석 요청**:
```
"@ohlollyday_orders_2601_2603.csv 랑 @ohlollyday_product_master.csv 같이 분석해줘.

1. 채널별 총 매출, 수수료, 원가, 영업이익 비교
2. 채널별 영업이익률 (매출 대비 순이익)
3. 월별 추이 (1월→3월, 채널별 매출 변화)
4. 주문 건당 평균 객단가 채널별 비교"
```

**후속 질문 예시**:
```
"카카오선물하기에서 이익률이 마이너스인 상품이 있어? 상품별로 채널 수익성 교차 분석해줘"
```

## 미션 2: 인스타 광고, 돈값 하고 있나?

> **썸머**: 2월부터 인스타 광고비를 꽤 올렸는데, 롤리 대표가 효과 있냐고 물어봤어. 나도 매주 월요일에 인사이트 데이터 정리하잖아. 도달이 늘었다는 건 아는데, 그게 실제 구매로 이어졌는지는 감으로만 알지 숫자로 증명 못 하겠어.
>
> **소니**: 나 릴스 만들 때 시간 엄청 들이거든. 새 포맷은 레퍼런스부터 찾아야 하고. 그런데 릴스가 캐러셀보다 나은 건지 아닌 건지 모르고 만들고 있어.

**분석 요청**:
```
"@ohlollyday_instagram_content_2601_2603.csv 분석해줘.

1. 콘텐츠 유형별(릴스/캐러셀/단일이미지/스토리) 도달 대비 전환율 비교
2. 광고 vs 오가닉 콘텐츠의 ROI 비교 (광고비 대비 주문금액)
3. 주제카테고리별 성과 차이 (어떤 주제가 구매를 가장 많이 만드는지)
4. 제작자별 콘텐츠 성과 비교"
```

**후속 질문 예시**:
```
"광고비 대비 매출이 가장 높은 콘텐츠 유형이 뭐야? ROAS(광고비 대비 매출) 기준으로"
```

## 미션 3: 인스타에서 자사몰까지, 파이프라인이 보여?

조이와 썸머가 각자 가진 데이터를 합치면 보이는 게 있다. 조이는 채널별 주문 데이터를, 썸머는 인스타 콘텐츠 성과 데이터를 갖고 있다. 두 사람이 매주 각각 2시간씩, 합치면 4시간을 쓰는데 한 번도 교차해서 본 적이 없다.

> **썸머**: 인스타 보고 자사몰 온 사람들이 어떤 상품을 사는지도 궁금해. 콘텐츠에서 보여준 상품을 그대로 사는 건지, 다른 걸 사는 건지.
>
> **조이**: 자사몰 유입경로에 '인스타그램'이 찍히는 주문이 있는데, 그게 콘텐츠 올린 날이랑 맞물리는지는 한 번도 안 봤어.

**분석 요청**:
```
"@ohlollyday_instagram_content_2601_2603.csv 랑 @ohlollyday_orders_2601_2603.csv 교차 분석해줘.

1. 인스타 콘텐츠 게시 후 자사몰 유입경로 '인스타그램' 주문 추이 (게시일 전후 비교)
2. 인스타 콘텐츠 유형별 자사몰 전환 퍼널 (도달 → 링크클릭 → 자사몰유입 → 주문)
3. 자사몰에서 유입경로 '인스타그램'인 주문의 상품/카테고리 분포
4. 월별 인스타 → 자사몰 전환율 변화 (광고비 증가 전후)"
```

**후속 질문 예시**:
```
"인스타 광고비를 월별로 합산하고, 같은 달 자사몰 인스타 유입 매출이랑 비교해서 월별 ROAS 추이 보여줘"
```

## 미션 4: Q2 채널 전략 제안서

> **에이미**: 핫트랙스에서 PART PART 노트 입점 확대 제안이 왔는데 수수료가 25%래. 이정원씨한테 물어봤더니 기존 매장에서도 재고 순환이 빠르지만은 않다고 하던데.
>
> **롤리**: 숫자 기반으로 판단하자. 나도 매일 아침 매출 데이터 확인하고 통장 잔고 보는데, 채널별로 실제로 남는 게 얼마인지는 직관적으로 안 보여. 채널 하나 더 늘릴 여력이 있는지, 있다면 어디에 집중할지 수요일까지 정리해줘.

**분석 요청**:
```
"세 개 파일 전부 종합해서 Q2 채널 전략을 정리해줘.

1. Q1 핵심 숫자 요약 (5줄 이내)
2. 채널별 Keep/Scale/Drop 판단 (근거 포함)
3. 인스타 광고 최적 예산 제안 (현재 ROAS 기반)
4. 상품 x 채널 최적 매칭 추천 (어떤 상품을 어디서 팔아야 마진이 좋은지)

간결하게. 수요일 팀미팅에서 의사결정 할 수 있는 형태로"
```

## 에필로그

**화요일 저녁 8시.**

숫자가 말해주는 게 있다. 카카오선물하기는 매출은 높지만 수수료 30%를 빼면 Happier Label 미니 파우치나 MNNS 마우스패드처럼 원가 비중이 높은 상품은 적자였다. 기프트박스처럼 마진율이 높은 상품만 수익이 났다. 반면 자사몰은 수수료가 3.5%뿐이라 거의 모든 상품에서 이익이 났고, 인스타 광고가 자사몰 유입을 2월부터 실제로 늘리고 있었다.

소니가 공들여 만든 릴스는 도달은 높았지만 실제 구매 전환은 캐러셀이 더 높았다. 링크를 넣기 쉬운 구조 때문이었다. 그리고 '신제품 출시'와 '제품 스타일링' 주제의 콘텐츠가 '일상 브이로그'보다 전환율이 3배 높았다.

매출 순위와 수익 순위는 달랐다. 그리고 그 차이를 만드는 건 수수료 구조였다.

**수요일 오전 10시. 성수동 사무실 미팅룸.**

> **뵤**: 카카오선물하기에서 파우치 팔면 남는 게 거의 없었다니... 정산할 때마다 이상하다 싶었는데 숫자로 보니까 확실하네. 기프트박스 위주로 가는 게 맞겠다.
>
> **썸머**: 릴스가 도달은 높은데 실제 구매 전환은 캐러셀이 더 높다는 거 처음 알았어. 소니야, 다음 주부터 캐러셀 비중 좀 올리자.
>
> **소니**: 아 진짜? 릴스에 시간을 엄청 쓰고 있었는데... 캐러셀이면 제작 시간도 더 짧아서 주 4개도 가능할 것 같아.
>
> **이정원**: 핫트랙스 PART PART 노트는 25% 수수료 빼도 마진이 나네요. 노트류만 입점 확대하는 거 어때요?
>
> **롤리**: 좋아. 채널 전략 분명해졌다. 자사몰 중심으로 가되 인스타 광고는 캐러셀에 집중. 카카오는 기프트세트만. 핫트랙스 입점 확대는 PART PART 노트류만.

**(조이, 속으로)**: 2시간 걸리던 정리를 30분 만에 끝내고, 숫자로 이야기하니까 결정이 이렇게 빨라지는구나.
