## 상황 설정

**월요일 아침 10시. 성수동 오롤리데이 사무실.**

조이는 모니터 앞에서 한숨을 쉬었다. 매주 월요일마다 반복되는 일이다. 자사몰 카페24 어드민, 네이버 스마트스토어 센터, 29CM 파트너 어드민, 카카오선물하기 파트너센터, 텐바이텐 셀러 페이지, 핫트랙스 정산 메일. 6개 채널의 숫자를 하나하나 엑셀에 옮겨 적는 데만 2시간이 걸린다.

그런데 오늘은 좀 다르다. 롤리 대표가 금요일에 슬랙에 남긴 메시지 때문이다.

> **롤리**: 이번 주 수요일에 Q2 채널 전략 정하자. 카카오선물하기 수수료가 30%인데 우리가 거기서 진짜 남는 게 있는 건지 한번 보고 싶어. 그리고 인스타 광고 2월부터 늘렸는데 자사몰 유입이 실제로 늘었는지도.

조이는 알고 있다. 채널이 6개나 되다 보니 "매출은 늘었는데 남는 돈은 비슷한" 현상이 계속되고 있다는 걸. 카카오선물하기는 설날과 발렌타인에 매출이 폭발하지만 수수료 30%에 카카오 전용 기프트박스 제작비까지 더하면 수익이 얼마나 되는지 아무도 정확히 모른다.

그리고 썸머가 2월부터 인스타그램 광고비를 월 200만원까지 늘렸는데, 그게 자사몰 매출로 이어졌는지도 감(感)으로만 안다. "인스타 보고 왔어요"라고 DM 오는 건 느는데, 숫자로는 증명 못 하고 있다.

**(속으로)**: 수요일까지 3일. 6개 채널의 진짜 수익성을 보여줘야 한다. 매출이 아니라 '남는 돈' 기준으로.

- 마감: 수요일 오전 10시 (롤리 대표 + 전체 팀미팅)
- 파일: 주문 1,920건, 상품 35개, 인스타 콘텐츠 50건

## 미션 1: 채널별 진짜 수익, 얼마나 남아?

> **뵤**: 조이씨, 나 3월 정산하면서 봤는데 카카오선물하기 수수료가 생각보다 많이 빠지더라. 매출 대비 마진이 얼마인지 채널별로 정리할 수 있어?

**분석 요청**:
```
"@orollidae_orders_2601_2603.csv 랑 @orollidae_product_master.csv 같이 분석해줘.

1. 채널별 총 매출, 수수료, 원가, 영업이익 비교
2. 채널별 영업이익률 (매출 대비 순이익)
3. 월별 추이 (1월→3월, 채널별 매출 변화)
4. 주문 건당 평균 객단가 채널별 비교"
```

**후속 질문 예시**:
```
"카카오선물하기에서 이익률이 마이너스인 상품이 있어? 상품별로 채널 수익성 교차 분석해줘"
```

## 미션 2: 인스타 광고, 돈값 하고 있나?

> **썸머**: 2월부터 인스타 광고비를 꽤 올렸는데, 롤리 대표가 효과 있냐고 물어봤어. 근데 나도 감으로만 알지 숫자로 증명은 못 하겠어...

**분석 요청**:
```
"@orollidae_instagram_content_2601_2603.csv 분석해줘.

1. 콘텐츠 유형별(릴스/캐러셀/단일이미지/스토리) 도달 대비 전환율 비교
2. 광고 vs 오가닉 콘텐츠의 ROI 비교 (광고비 대비 주문금액)
3. 주제카테고리별 성과 차이 (어떤 주제가 구매를 가장 많이 만드는지)
4. 제작자별 콘텐츠 성과 비교"
```

**후속 질문 예시**:
```
"광고비 대비 매출이 가장 높은 콘텐츠 유형이 뭐야? ROAS(광고비 대비 매출) 기준으로"
```

## 미션 3: 인스타에서 자사몰까지, 파이프라인이 보여?

썸머와 조이가 각자 가진 데이터를 합치면 보이는 게 있다. 인스타그램 콘텐츠가 자사몰 매출로 이어지는 경로.

> **소니**: 나 릴스 만들 때 제품 태그 넣는 게 귀찮아서 가끔 빼는데, 태그 있는 거랑 없는 거랑 차이가 있어?
>
> **조이**: 아 그리고 인스타 보고 자사몰 온 사람들이 어떤 상품을 사는지도 궁금해. 인스타에서 보여준 상품을 그대로 사는 건지, 다른 걸 사는 건지.

**분석 요청**:
```
"@orollidae_instagram_content_2601_2603.csv 랑 @orollidae_orders_2601_2603.csv 교차 분석해줘.

1. 인스타 콘텐츠 게시 후 자사몰 유입경로 '인스타그램' 주문 추이 (게시일 전후 비교)
2. 인스타 콘텐츠 유형별 자사몰 전환 퍼널 (도달 → 링크클릭 → 자사몰유입 → 주문)
3. 자사몰에서 유입경로 '인스타그램'인 주문의 상품/카테고리 분포
4. 월별 인스타 → 자사몰 전환율 변화 (광고비 증가 전후)"
```

**후속 질문 예시**:
```
"인스타 광고비를 월별로 합산하고, 같은 달 자사몰 인스타 유입 매출이랑 비교해서 월별 ROAS 추이 보여줘"
```

## 미션 4: Q2 채널 전략 제안서

> **에이미**: 핫트랙스에서 PART PART 노트 입점 제안이 왔는데 수수료가 25%래. 들어가야 하나?
>
> **롤리**: 숫자 기반으로 판단하자. 채널 하나 더 늘릴 여력이 있는지, 있다면 어디에 집중할지.

**분석 요청**:
```
"세 개 파일 전부 종합해서 Q2 채널 전략을 정리해줘.

1. Q1 핵심 숫자 요약 (5줄 이내)
2. 채널별 Keep/Scale/Drop 판단 (근거 포함)
3. 인스타 광고 최적 예산 제안 (현재 ROAS 기반)
4. 상품 x 채널 최적 매칭 추천 (어떤 상품을 어디서 팔아야 마진이 좋은지)

간결하게. 수요일 팀미팅에서 의사결정 할 수 있는 형태로"
```

## 에필로그

**화요일 저녁 8시.**

숫자가 말해주는 게 있다. 카카오선물하기는 매출은 높지만 수수료 30%를 빼면 키링이나 기프트박스처럼 마진율이 높은 상품만 수익이 났다. 반면 자사몰은 수수료가 3.5%뿐이라 거의 모든 상품에서 이익이 났고, 인스타 광고가 자사몰 유입을 실제로 늘리고 있었다.

매출 순위와 수익 순위는 달랐다. 그리고 그 차이를 만드는 건 수수료 구조였다.

**수요일 오전 10시. 성수동 사무실 미팅룸.**

> **뵤**: 카카오선물하기에서 파우치 팔면 남는 게 거의 없었다니... 기프트박스 위주로 가는 게 맞겠다
>
> **썸머**: 릴스가 도달은 높은데 실제 구매 전환은 캐러셀이 더 높다는 거 처음 알았어. 콘텐츠 믹스 바꿔야겠다
>
> **롤리**: 좋아. 채널 전략 분명해졌다. 자사몰 중심으로 가되 인스타 광고는 캐러셀에 집중. 카카오는 기프트세트만. 핫트랙스 신규 입점은 노트류만.

**(조이, 속으로)**: 2시간 걸리던 정리를 30분 만에 끝내고, 숫자로 이야기하니까 결정이 이렇게 빨라지는구나.
