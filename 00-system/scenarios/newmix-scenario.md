## 상황 설정

**금요일 오후 3시. 안국.**

사무실 5층 자리에서 기획 업무를 마무리하고 있다. 창밖으로 창덕궁이 보인다. 오늘도 성수 매장에서 외국인 관광객들이 줄을 서고 있다는 슬랙이 올라왔다.

슬랙 알림이 울린다.

> **규림님**: 원님, 봉진님이 월요일 오전에 2월 실적 리뷰하고
> 3월 벚꽃 시즌 전략 논의하자고 하셨어요
>
> 성수뿐 아니라 북촌이랑 온라인도 같이 봐주세요
> 전채널 매출 데이터랑 재고 현황 분석해서
> 인사이트 정리해주실 수 있어요?
>
> 봉진님 스타일 아시잖아요
> 숫자 나열하면 "그래서 뭐?" 하시고
> 단순 보고서는 넘기시고 스토리가 있어야 넘어가요
>
> 특히 외국인 관광객 데이터 중요해요
> 벚꽃 시즌에 인바운드 어떻게 대응할지 근거가 필요하거든요
> 페이히어 POS에서 외국인 카드 결제 시 국적 코드가 자동으로 잡혀요

**(속으로)**: 봉진님... 빈틈 보이면 바로 파고드시는데. 준비 확실히 해야 한다.

슬랙이 또 온다.

> **규림님**: 아 그리고 저도 하나만 부탁드려요
> 일본 고객이 어떤 맛 많이 사는지 알 수 있을까요?
> SNS 콘텐츠에 반영해야 하거든요
> 국적별로 인기 상품 다르면 좋겠어요

> **은지님**: 저도요! 3월 벚꽃 시즌 한정 기획하고 있는데
> 지금 제일 잘 나가는 맛이 뭔지 알아야 해요
> 한정판 구성 짤 때 참고할게요
> 아 그리고 온라인 매출도 봐야 해요
> 네이버랑 쿠팡이랑 컬리 중에 어디가 제일 잘 되고 있는지

> **새봄님**: 패키지 디자인 시안 준비하는데
> 북촌 매장이 성수랑 고객층이 어떻게 다른지 궁금해요
> 패키지 디자인을 매장별로 다르게 가야 할지...

**(속으로)**: 모두가 데이터를 원한다... 그런데 파일 다섯 개를 언제 다 보지?

- 진짜 마감: 월요일 오전 10시 전략 미팅
- 체감 마감: 일요일 저녁 (규림님 검토 시간 필요)
- 실제 가용 시간: 지금부터 내일 오후
- 받은 파일: 성수 1~2월 매출, 북촌 2월 매출, 온라인 2월 매출, 재고 현황

**Claude Code가 없었다면?**
- 엑셀 열어서 파일 5개 피벗 테이블 만들기 (2시간)
- 국적별 필터링하고 수식 넣기 (1시간)
- 매장간 비교 테이블 정리 (1시간)
- 온라인 채널별 분석 (1시간)
- 재고 데이터 크로스 체크 (30분)
- 차트 그리기 (1시간)
- 인사이트 정리하고 보고서 초안 (1시간 30분)
- 총 **8시간+, 주말 반납 확정**

**Claude Code가 있다면?**
- 대화로 분석 요청 (30분)
- 결과 확인하고 추가 질문 (30분)
- 총 **1시간, 오늘 정시 퇴근 가능**

자, 시작해보자.

## 미션 1: 성수 매출 현황 + 1월 대비 성장

규림님의 첫 번째 요청. 봉진님이 제일 먼저 물어볼 것.

> **규림님**: 봉진님이 먼저 물어보실 거예요
> "2월 성수 어땠어?" 이거거든요
>
> 단순히 "매출 얼마입니다"는 안 돼요
> "어떤 고객이, 뭘, 얼마나 샀는지" 흐름이 있어야 해요
>
> 1월 데이터도 있으니까 전월 대비 비교해주세요

**(속으로)**: 일단 전체 그림부터 보자. 봉진님은 숫자 하나라도 앞뒤가 안 맞으면 바로 짚으시니까.

**분석 요청**:
```
"@newmix_sales_seongsu_2602.csv 랑 @newmix_sales_seongsu_2601.csv 두 파일 분석해줘.
@sales-growth-framework.md 관점으로
2월 매출 현황 전체 정리하고, 1월 대비 성장률도 비교해줘.
주차별 추이, 카테고리별 매출, 베스트셀러 TOP 5까지 한 번에 봐줘"
```

**결과 확인 후 추가 질문 예시**:
```
"봉진님 보고용으로 핵심 인사이트 3줄로 요약해줘. 숫자 나열 말고 스토리로"
```

## 미션 2: 고객 국적별 분석

미션 1 끝내고 나니 슬랙이 두 개 와 있다.

> **규림님**: 원님, 아까 부탁드린 거 어떻게 되고 있어요?
> 일본 고객 데이터 볼 수 있을까요?
>
> 우리 인스타 일본어 콘텐츠 만들려고 하거든요
> 어떤 맛이 인기인지 알면 콘텐츠 방향 잡기 좋을 것 같아요

> **은지님**: 저도 궁금해요! 대만 고객이 요즘 많이 오는 것 같은데
> 혹시 국적별로 다르게 사는 거 있을까요?
> 벚꽃 시즌 기획에 반영할게요

**(속으로)**: 마침 매출 데이터에 고객 국적이 있네. "사소한 것을 위대하게" 만드는 게 뉴믹스 아닌가.

**분석 요청**:
```
"@newmix_sales_seongsu_2602.csv 에서 고객 국적별 분석해줘.
@tourism-segmentation-framework.md 관점으로
국적별 매출 비중, 인기 상품 차이, 평균 객단가 비교하고,
외국인이 주로 오는 시간대가 한국 고객이랑 다른지도 봐줘.
특히 일본, 대만, 미국 고객 비교가 필요해"
```

**결과 확인 후 추가 질문 예시**:
```
"일본 고객한테 인스타 콘텐츠 만들려면 어떤 상품을 강조해야 할까?"
```

## 미션 3: 재고 관리 & 벚꽃 시즌 준비

규림님과 은지님한테 답장을 보내고 있는데, 규림님에게 또 슬랙이 온다.

> **규림님**: 아 그리고 중요한 거 하나 더요
> 물류팀에서 오리지날 커피포 재고 바닥이라고 연락 왔어요
>
> 오리지날이 우리 베스트셀러인데 이거 품절되면 큰일이에요
> 3월 벚꽃 시즌에 관광객 더 늘어날 텐데...
>
> 재고 현황 전체적으로 한번 봐주시겠어요?
> 긴급 발주 필요한 것들 정리해주세요
>
> 봉진님이 "왜 품절이야?" 물으시면 답변할 수 있게 준비해야 해요

**(속으로)**: 베스트셀러가 품절 직전이라고? 이건 진짜 급하다.

**분석 요청**:
```
"@newmix_inventory_data.csv 재고 현황 전체 분석해줘.
@inventory-analysis-framework.md 관점으로
재고일수 7일 미만 긴급 품목 우선으로 정리하고,
카테고리별 상태 요약도 해줘.
3월 벚꽃 시즌에 관광객 30% 증가 가정하면 지금 재고로 며칠 버틸 수 있는지도"
```

**결과 확인 후 추가 질문 예시**:
```
"긴급 발주 필요한 품목을 우선순위별로 정리해서 발주 리스트 만들어줘"
```

## 미션 4: 성수 vs 북촌 매장 비교

벌써 4시 반. 생각보다 빠르게 진행되고 있다.

> **(속으로)**: 이 정도면 오늘 안에 끝낼 수 있겠는데?

근데 새봄님한테 슬랙이 온다.

> **새봄님**: 원님 감사해요! 그런데 하나만 더 부탁드려도 될까요?
> 북촌 매장이 성수랑 고객층이 어떻게 다른지 궁금해요
>
> 패키지 디자인을 매장별로 다르게 가야 할지
> 아니면 통일해도 되는지 판단하려면
> 매출 데이터 비교가 필요하거든요
>
> 특히 시나몬약과맛이 북촌에서만 파는 건데
> 성과가 어떤지도 궁금해요
>
> 벚꽃 시즌 매장별 기획에 반영할게요!

**(속으로)**: 성수 vs 북촌 비교 분석이네. 두 매장의 성격 차이를 데이터로 보여주면 되겠다.

**분석 요청**:
```
"@newmix_sales_seongsu_2602.csv 랑 @newmix_sales_bukchon_2602.csv 비교 분석해줘.
@store-comparison-framework.md 관점으로
매출, 주문 수, 객단가 차이 보고, 국적 구성이 어떻게 다른지,
매장별 베스트셀러도 비교해줘.
시나몬약과맛 같은 북촌 한정 상품 성과도 궁금해"
```

**결과 확인 후 추가 질문 예시**:
```
"이 데이터 기준으로 패키지 디자인을 매장별로 다르게 가야 할까, 통일해도 될까?"
```

## 미션 5: 온라인 채널 분석

> **은지님**: 원님, 온라인 매출도 봐야 해요
> 네이버랑 쿠팡이랑 컬리 중에 어디가 제일 잘 되고 있는지...
>
> 벚꽃 시즌 프로모션 채널 정해야 하거든요
> 전 채널에 다 뿌리면 비용이 너무 크니까
> 집중할 곳을 정해야 해요

**(속으로)**: 온라인 데이터까지 보면 전 채널 그림이 완성되겠다.

**분석 요청**:
```
"@newmix_online_sales_2602.csv 분석해줘.
@channel-mix-framework.md 관점으로
네이버, 쿠팡, 컬리 채널별 매출 비교하고, 채널마다 인기 상품이 다른지 봐줘.
배송지역 기준으로 어느 지역에서 주문이 많은지도 정리하고,
오프라인(성수+북촌) 매출이랑 비교해서 온오프라인 비중도 알려줘"
```

**결과 확인 후 추가 질문 예시**:

## 미션 6: 봉진 의장 보고용 전채널 종합 보고서

시계를 본다. **5시 10분.**

보통 금요일 이 시간이면 퇴근 준비를 하고 있을 텐데... 근데 이상하게 기분이 좋다.

> **(속으로)**: 이거 진짜 되네? 평소 같았으면 아직 엑셀 필터 걸고 있었을 텐데.

규림님에게 슬랙이 온다.

> **규림님**: 원님, 어디까지 되셨어요?
>
> 아 벌써 그렇게까지요? 빠르시다
>
> 그러면 마지막으로 이것만 부탁드려요
> 지금까지 분석한 거 1페이지로 요약해주세요
> 성수, 북촌, 온라인 전채널 종합으로요
>
> 봉진님 스타일 아시잖아요
> - 핵심 숫자 3개
> - "그래서 뭐?" 에 대한 답 3개
> - 3월 벚꽃 시즌 액션 아이템 3개
>
> 이 포맷으로 정리해주시면 제가 자료 만들게요
>
> 아 그리고 차트 하나만 넣어주세요
> 봉진님이 비주얼 좋아하셔요 (디자이너 출신이시잖아요)

**(속으로)**: 맞지, 봉진님은 디자인 전공하신 분이라 비주얼이 중요하다. 대충 만들면 바로 티난다.

**분석 요청**:
```
"지금까지 분석한 5개 파일 전부 종합해서 봉진 의장님 보고용 1페이지 요약본 만들어줘.
@executive-report-framework.md 관점으로
포맷은 이거야:
- 핵심 숫자 3개 (전채널 합산 매출, 1월 대비 성장률, 외국인 비중)
- '그래서 뭐?'에 대한 답 3개
- 3월 벚꽃 시즌 액션 아이템 3개 (매장별, 온라인 각각)
차트도 하나 넣어줘. 봉진님이 디자이너 출신이라 비주얼이 중요해"
```

**결과 확인 후 추가 질문 예시**:
```
"재고 위험 품목도 빨간색으로 표시한 표로 따로 만들어줘"
```
