# Claude Code 시연 가이드 - 무신사/29CM 마케팅

> **용도**: 마케팅 리더십 미팅 시 라이브 시연용
> **시연 시간**: 20-25분 (미션 2개 + 피날레)

---

## 시연 전 참고

### 이 시연의 목적

마케팅 리더십이 궁금한 건 "도구가 뭘 하나"가 아니라 **"우리 팀이 어떻게 달라지나"**입니다.

보여줘야 할 것:
1. **탐색의 속도** -- 데이터에 자연어로 질문하고, 후속 질문을 계속 던지는 루프
2. **분석 → 실행 연결** -- 숫자 정리에서 끝나지 않고 카피/기획까지 이어지는 흐름
3. **프레임워크 효과** -- 같은 데이터도 관점을 주면 깊이가 달라진다는 것

### Tableau/GA4와 뭐가 다른가?

| | Tableau/Looker/GA4 | Claude Code |
|---|---|---|
| 질문 방식 | 미리 만든 대시보드에서 필터 | 자연어로 아무 질문이나 |
| 후속 질문 | 새 차트 만들어야 함 | "왜?" 한마디면 됨 |
| 분석 + 글쓰기 | 별도 도구 필요 | 한 자리에서 보고서까지 |
| 예상 못 한 질문 | 대시보드에 없으면 끝 | 즉석에서 파고들기 |

시연 중 이 차이가 자연스럽게 드러나도록 **"후속 질문"을 즉석에서 던지는 장면**을 반드시 보여주세요.

## 상황 설정

**패션 이커머스 마케팅팀의 월간 리뷰.**

매달 마케팅 리더십이 모여서 전월 성과를 리뷰하고 다음 달 방향을 잡습니다.
이번 달 안건:
- 2월 인플루언서 협업은 어땠는지
- 자체 SNS 채널은 어떤 콘텐츠가 먹혔는지
- 3월 S/S 시즌에 뭘 다르게 해야 하는지

팀원이 데이터를 엑셀로 정리해서 올려놨는데, 리뷰 미팅까지 시간이 없습니다.

**지금부터 Claude Code로 20분 만에 리뷰 자료를 만듭니다.**

---

## 미션 1: 인플루언서 협업 성과 리뷰 (8-10분)

마케팅 리더십이 실제로 하는 질문으로 시작합니다.

### Step 1: 먼저 데이터를 있는 그대로 봅니다

```
"@musinsa/musinsa_influencer_2602.csv 2월 인플루언서 협업 전체 현황 정리해줘.
총 몇 건 협업했고, 총 비용 얼마 썼고, 등급별로 어떻게 분포되어 있는지.
플랫폼별 비중도 같이."
```

> **시연 포인트**: 400행 데이터를 10초 만에 요약. "일단 전체 그림 먼저 보자"

### Step 2: 프레임워크를 더해서 깊이를 만듭니다

```
"이 데이터를 @influencer-effectiveness-framework.md 관점으로 다시 봐줘.

등급별 Engagement Rate랑 비용 대비 전환 효율을 비교하고,
인플루언서 티어 매트릭스로 시각화해줘.
X축 비용, Y축 전환수, 버블 크기 노출수, 색상 등급별로."
```

> **시연 포인트**: 같은 데이터인데 프레임워크 하나 추가했을 뿐인데 분석 깊이가 달라짐.
> "어떤 관점으로 보느냐"가 분석의 질을 결정한다는 것을 보여줍니다.

### Step 3: 후속 질문 -- 이게 핵심입니다

여기서 **즉석으로 질문을 던집니다**. 미리 정해진 대시보드와 다른 점이 여기서 드러납니다.

```
"그러면 카테고리별로 쪼개서 봐줘.
아우터 카테고리에서는 어떤 등급이 효율 좋고, 스니커즈는 다를 수 있잖아"
```

또는 청중의 실제 질문이 나오면 그걸 그대로 던집니다:
- "29CM 쪽 인플루언서만 따로 보면?"
- "틱톡이랑 인스타 효율 차이는?"
- "비용 대비 저장수가 높은 인플루언서 TOP 10은?"

> **시연 포인트**: 청중이 "이것도 돼?"하면서 질문을 던지기 시작하면 성공.
> Tableau에서는 새 차트를 만들어야 하는 질문을, 자연어 한 줄로 즉시 답합니다.

---

## 미션 2: SNS 콘텐츠 성과 + 실행까지 (8-10분)

### Step 1: 콘텐츠 성과 분석

```
"@musinsa/musinsa_sns_organic_2602.csv 를 @content-performance-framework.md 관점으로 분석해줘.

콘텐츠 유형별(릴스/피드/카루셀/스토리/쇼츠) 도달, ER, Save Rate 비교하고,
두 계정(@musinsa_official vs @29cm_official) 성과 비교도 해줘.
Rate 기준으로 공정하게."
```

> **시연 포인트**: 팔로워 수가 다른 두 계정을 Rate로 비교. 29CM이 작지만 engagement가 더 높다는 패턴이 나옴 -- 이건 "양보다 질"이라는 이 팀의 실제 고민과 연결됩니다.

### Step 2: 분석에서 실행으로 -- 이 전환이 차별점

```
"지금 분석 결과 기반으로,
3월 S/S 시즌 인스타 릴스용 캠페인 카피를 5개 만들어줘.

조건:
- 29CM 톤앤매너 (감성적, 에디토리얼, '발견'과 '영감')
- 분석에서 나온 인사이트 반영
- 저장 유도 CTA 포함 (Save Rate 높이는 방향)
- 타깃: 2030 여성, 봄 스타일링"
```

> **시연 포인트**: 이게 Tableau/GA4에서는 절대 안 되는 것.
> 데이터 분석 → 인사이트 → 실행(카피) 까지 한 흐름.
> "아까 분석에서 카루셀 저장이 높다고 했으니, 이 카피를 카루셀로도 만들어줘" 같은 연결.

### Step 3: 후속 질문

```
"무신사 톤으로도 변형해줘. 좀 더 직접적이고 실용적으로."
```

> **시연 포인트**: 같은 인사이트인데 브랜드 톤에 따라 다른 결과물. 멀티브랜드 운영팀에게 실용적.

## 피날레: 리뷰 자료 정리 (3-5분)

```
"지금까지 분석한 내용 전부 종합해서 마케팅 리뷰 미팅용 1페이지 요약 만들어줘.

핵심 숫자 3개, 인사이트 3개, 3월 액션 3개.
차트 포함해서 HTML로 만들어줘."
```

> **시연 포인트**: "이걸 20분 만에 했다"는 체감.
> 엑셀 정리 + 차트 + 인사이트 + 보고서 = 보통 하루 이상 걸리는 작업.

## 시연 중 유의사항

### 반드시 보여줘야 할 것
- **후속 질문 루프**: 미리 준비된 시나리오만 따라가지 말고, 청중 반응에 따라 즉석 질문
- **프레임워크 전/후 비교**: 같은 데이터에 프레임워크 넣기 전/후 차이
- **분석 → 실행 전환**: 숫자에서 카피/기획으로 넘어가는 순간

### 하지 말아야 할 것
- 인사이트를 "발견"처럼 포장하지 않기 -- 팀장급은 이미 감으로 알고 있음. "감으로 알던 걸 데이터로 확인하고, 거기서 한 걸음 더 파고들 수 있다"로 프레이밍
- "엑셀 대체"로만 포지셔닝하지 않기 -- "탐색적 대화가 가능한 분석 파트너"
- 결과가 완벽한 척하지 않기 -- 숫자가 이상하면 "이 부분은 확인이 필요하네요, 조건을 바꿔볼게요" 하고 수정하는 과정을 보여주면 오히려 신뢰가 올라감

### 시간 배분
| 구간 | 시간 | 내용 |
|------|------|------|
| 미션 1 | 8-10분 | 인플루언서 현황 → 프레임워크 → 후속질문 |
| 미션 2 | 8-10분 | SNS 분석 → 카피 생성 → 톤 변형 |
| 피날레 | 3-5분 | HTML 요약 리포트 |
| 버퍼 | 2-3분 | 청중 질문 대응 |
