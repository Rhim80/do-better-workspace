# 채널-콘텐츠 통합 분석 프레임워크

## 분석 구조

### 1단계: 인스타 → 자사몰 파이프라인
- 인스타 콘텐츠 성과 데이터
- 자사몰 유입경로 '인스타그램' 주문 데이터
- 시간축 연결 (게시일 → 유입 → 구매)

### 2단계: 채널별 유입경로 분석
- 자사몰: 인스타그램/블로그검색/직접유입/카카오채널
- 각 유입경로별 객단가, 구매 상품, 재구매율

### 3단계: 크로스 채널 효과
- 인스타 콘텐츠가 자사몰 외 다른 채널에도 영향을 주는가
- 월별 인스타 활동량 vs 전체 채널 매출 상관관계

### 4단계: 최적 배분 전략
- 채널별 마진율 + 유입경로 효과 종합
- "인스타 광고 → 자사몰 유입"이 가장 효율적인 경로인가
- 채널/콘텐츠/상품 최적 조합

## 핵심 질문

1. 인스타 광고비가 자사몰 매출로 얼마나 전환되는가?
2. 어떤 콘텐츠 유형이 실제 구매를 가장 많이 만드는가?
3. 자사몰 외 채널(네이버, 29CM)에도 인스타 효과가 있는가?
4. 최적 채널 전략: 어디에 집중하고, 무엇을 줄여야 하는가?
