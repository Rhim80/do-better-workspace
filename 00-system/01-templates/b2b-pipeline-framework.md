## 1. 파이프라인 퍼널 (Pipeline Funnel)

**출처**: Skok, D. (2010). "SaaS Metrics 2.0". For Entrepreneurs Blog / HBS Working Knowledge
**적용**: SaaS 파이프라인을 콘텐츠 비즈니스에 맞게 변형

### 트래블코드 파이프라인

```
콘텐츠 노출 → 트립 문의 → 트립 참가 → 컨설팅 문의 → 컨설팅 계약
```

### 단계��� 전환율

| 단계 | 계산 | 의미 |
|------|------|------|
| 콘텐츠 → 트립 문의 | 총 문의 / 총 콘텐츠 도달 | 콘텐츠 전환 효율 |
| 트립 문의 → 참가 | 참가자 / 문의 건수 | 영업 전환율 |
| 참가 → 컨설팅 전환 | 컨설팅 건수 / 참가자 수 | 후속 전환율 |

## 2. Trip Total Value (TTV)

**개념**: 트립 1회의 가치 = 직접 매출 + 후속 컨설팅 매출

```
TTV = 트립 참가비 매출 + 연결된 컨설팅 계약 금액

예시:
  상하이 리테일 트렌드 투어 (TRIP-2601-02)
  직접 매출: 1,500만원 (B2B 계약)
  후속 컨설팅: 2,500만원 (위시컴퍼니 콘텐츠 기획)
  TTV = 4,000만원
```

### TTV가 중요한 이유

트립만 놓고 보면 소규모 매출이지만, 후속 컨설팅까지 합산하면 ���치가 달라진다. 이걸 모르면 "트립은 수고 대비 매출이 적다"는 잘못된 판단을 할 수 있다.

## 3. 유입경로별 파이프라인 효율

### 경로 유형

| 유입경로 | 특성 | 기대 전환 |
|----------|------|-----------|
| 인사이트트립 | 직접 경험 후 전환 | 높음 (신뢰 형성) |
| 기존고객 | 재계약/확장 | 매우 높음 |
| 뉴스레터 | 콘텐츠 → 직접 전환 | 중간 |
| 도서 | 브랜드 인지 → 전환 | 낮지만 고품질 |
| 지인추천 | 관계 기반 | 높음 |

### 경로별 평균 계약 금액 비교

각 유입경로의 평균 계약 금액을 비교하면 "어디에 마케팅 리소스를 투입해야 하는가"의 답이 나온다.

## 4. 전환 타이밍 분석

**핵심**: 트립 참가 → 컨설팅 계약까지 걸리는 기간

```
전환 리드타임 = 컨설팅 시작일 - 트립 종료일
```

### 해석

| 리드타임 | 의미 | 액션 |
|----------|------|------|
| 1개�� 이내 | 즉각 전환 (니즈가 명확했음) | 트립 중 컨설팅 안내 |
| 1-3개월 | 검토 후 전환 | 후속 팔로업 시점 |
| 3개월+ | 장기 잠재 | 뉴스레터/콘텐츠로 nurture |
