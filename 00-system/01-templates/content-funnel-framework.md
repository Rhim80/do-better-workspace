# 콘텐츠 퍼널 분석 프레임워크

> 기반: AARRR Funnel (McClure, 2007), Content Attribution, Channel Shift Analysis

---

## 1. 콘텐츠 퍼널 (Content → Conversion)

**출처**: Dave McClure (2007). "Startup Metrics for Pirates: AARRR!". 500 Startups
**발전**: Content Marketing Institute (2023). B2B Content Marketing Benchmarks

**핵심**: 콘텐츠의 가치를 '좋아요'가 아니라 **비즈니스 전환**으로 측정

### 퍼널 단계

```
콘텐츠 발행 → 도달/노출 → 참여(오픈/좋아요) → 클릭 → 전환(문의/신청)
```

### 채널별 핵심 지표

| 단계 | 뉴스레터 | 인스타그램 |
|------|----------|-----------|
| 도달 | 발송수 | 도달수 |
| 참여 | 오픈수 (오픈율) | 좋아요+댓글+저장 |
| 클릭 | 링크클릭 | 링크클릭+프로필방문 |
| 전환 | 트립문의건수 | 트립문의건수 |

## 2. 채널 시프트 분석 (Channel Shift Analysis)

**핵심**: 채널 비중을 변경했을 때, 전체 전환 총량이 유지/증가/감소했는지 측정

### 분석 방법

```
피벗 전 (1월):
  뉴스레터 전환 + 인스타 전환 = 총 전환 A

피벗 후 (3월):
  뉴스레터 전환 + 인스타 전환 = 총 전환 B

채널 시프트 효과 = B - A
  양수: 피벗이 전환 총량을 늘렸다
  음수: 피벗으로 전환이 줄었다
  0: 채널만 바뀌고 총량은 동일
```

### 주의점

- **빈도 보정**: 월별 콘텐츠 건수가 다르면 "건당 전환율"로 비교
- **계절성**: 트립 모집 시즌(1월)과 비시즌의 차이 고려
- **카니발리제이션**: 인스타 증가분이 뉴스레터 감소분을 단순 대체한 건 아닌지

## 3. 도시별 콘텐츠 효율 (Content Efficiency by Topic)

**출처**: Kaushik, A. (2010). Web Analytics 2.0. Sybex - "Content Consumption → Amplification → Conversion" 모델

### 분석 매트릭스

| | 전환 높음 | 전환 낮음 |
|---|----------|----------|
| **참여 높음** | Star 콘텐츠 (확대) | 엔터테인먼트 (전환 경로 점검) |
| **참여 낮음** | Hidden Gem (도달 확대) | 재검토 (축소 또는 폐기) |

### 적용

- X축: 참여율 (오픈율 또는 engagement rate)
- Y축: 건당 트립 문의 수
- 도시별로 찍어서 사분면에 배치
- Star 도시에 콘텐츠 비중 확대

---

## 4. 콘텐츠 유형별 효율 (Format Effectiveness)

### 인스타그램 포맷 비교 기준

| 지표 | 계산 | 의미 |
|------|------|------|
| 도달 효율 | 평균 도달수 | 얼마나 많이 보이는가 |
| 참여 효율 | (좋아요+댓글+저장) / 도달수 | 본 사람 중 반응한 비율 |
| 전환 효율 | 트립문의 / 도달수 x 1000 | 1,000 도달당 문의 수 |

**핵심**: 릴스는 도달이 높지만 전환이 낮을 수 있고, 카드뉴스는 도달은 낮지만 전환이 높을 수 있다. 목적에 따라 포맷을 선택.

---

## 분석 순서 요약

1. **전체 현황**: 채널별 콘텐츠 건수, 총 전환 수
2. **채널 시프트**: 피벗 전후 총량 비교
3. **도시별 효율**: 사분면 매트릭스로 우선순위
4. **포맷별 효율**: 전환 효율 기준으로 최적 포맷 식별
5. **제안**: 채널 비중 + 도시 우선순위 + 포맷 권고
