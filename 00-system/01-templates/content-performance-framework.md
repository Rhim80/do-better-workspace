# 콘텐츠 퍼포먼스 분석 프레임워크

> 기반: Social Media Analytics, Content Marketing Institute Benchmarks, Platform Algorithm Insights

---

## 1. 핵심 성과 지표

**출처**: Sprout Social Index 2024, Hootsuite Digital Trends 2024

### 콘텐츠 KPI

| KPI | 공식 | 의미 |
|-----|------|------|
| Engagement Rate | (좋아요+댓글+저장+공유) / 노출수 x 100 | 콘텐츠 반응도 |
| Reach Rate | 노출수 / 팔로워수 x 100 | 콘텐츠 도달력 |
| Save Rate | 저장수 / 노출수 x 100 | 구매 고려 (관심 저장) |
| Share Rate | 공유수 / 노출수 x 100 | 바이럴 잠재력 |
| Follower Growth | 일간 팔로워 증감 | 계정 성장성 |
| Profile Visit Rate | 프로필방문 / 노출수 x 100 | 브랜드 탐색 유도 |

### 반응의 깊이 위계

```
공유 (Share)    ← 가장 깊은 반응: "다른 사람에게도 보여주고 싶다"
  ↑
저장 (Save)     ← 깊은 반응: "나중에 다시 보고 싶다 / 구매 고려"
  ↑
댓글 (Comment)  ← 중간 반응: "의견을 남길 만큼 관여"
  ↑
좋아요 (Like)   ← 얕은 반응: "괜찮네" (습관적 가능)
  ↑
노출 (Reach)    ← 최소 반응: 봤다

핵심: 좋아요 수보다 저장+공유 수가 콘텐츠 품질의 진짜 지표
```

## 2. 콘텐츠 유형별 분석

**출처**: Later Social Media Benchmarks 2024, Instagram @creators

### 유형별 성과 특성

| 유형 | 강점 지표 | 약점 지표 | 최적 활용 |
|------|-----------|-----------|-----------|
| 릴스 | 도달 (Reach) | 저장 | 신규 유입, 인지도 확대 |
| 피드 | 저장 (Save) | 도달 | 상품 상세, 스타일링 |
| 카루셀 | 저장+ER | 도달 | 큐레이션, 가이드, 비교 |
| 스토리 | 즉시성 | 지속성 | 이벤트 공지, 일상 |
| 쇼츠 | 도달 (유튜브/틱톡) | 저장 | 트렌드 참여, 바이럴 |

### 유형별 벤치마크

```
도달 (팔로워 대비 Reach Rate)
릴스:   200~500%  ← 알고리즘 추천으로 팔로워 이상 도달
쇼츠:   150~400%
카루셀: 80~150%
피드:   50~100%
스토리: 30~60%

Engagement Rate
카루셀: 8~15%  ← 스와이프 행동 자체가 engagement
릴스:   6~12%
쇼츠:   7~14%
피드:   5~10%
스토리: 2~5%

Save Rate
카루셀: 3~6%   ← 정보형 콘텐츠, "나중에 참고"
피드:   1.5~4%
릴스:   0.8~2%
쇼츠:   0.5~1.5%
```

## 5. 계정 비교 분석 (멀티 브랜드)

### 비교 프레임워크

```
같은 그룹 내 계정 비교 시:
1. 절대 수치 비교: 팔로워 크기가 다르면 불공정 → Rate(비율)로 비교
2. ER로 비교: 커뮤니티 '질' 측정
3. 콘텐츠 믹스 비교: 어떤 유형을 많이 올리는지
4. 테마 반응 비교: 같은 테마에 어떤 계정이 더 반응 좋은지
5. 성장률 비교: 팔로워 증가 속도

인사이트 패턴:
- 팔로워 작은 계정이 ER 높음 → 충성 커뮤니티 (29CM 패턴)
- 팔로워 큰 계정이 도달 높음 → 인지도 채널 (무신사 패턴)
- 두 계정은 "역할이 다르다"가 결론일 수 있음
```

## 6. 콘텐츠 포트폴리오 밸런스

### 3:4:3 룰

```
30% 인지 콘텐츠 (Awareness)
  - 릴스, 쇼츠, 트렌드 참여
  - 목표: 도달 극대화, 신규 팔로워

40% 고려 콘텐츠 (Consideration)
  - 카루셀, 피드, 에디토리얼
  - 목표: 저장, 상품 탐색 유도

30% 전환 콘텐츠 (Conversion)
  - 이벤트, 프로모션, 스토리 스와이프업
  - 목표: 구매, 사이트 방문

진단: 현재 콘텐츠 믹스가 어디에 치우쳐 있는지?
→ 인지 과다: 도달은 높은데 전환 약함
→ 전환 과다: 세일/이벤트만 올리면 팔로워 이탈
→ 고려 과다: 좋은 콘텐츠인데 성장이 정체
```

## 7. 실습 적용 예시

### 프롬프트 템플릿

```
"[SNS 데이터]를 분석해줘.

@content-performance-framework.md 관점으로:
- 콘텐츠 유형별(릴스/피드/카루셀/스토리/쇼츠) Engagement Rate, Reach Rate, Save Rate 비교
- 테마별(룩북/상품소개/이벤트/브랜드스토리) 반응 차이
- 계정별 성과 비교 (Rate 기준으로 공정하게)
- 현재 콘텐츠 포트폴리오 밸런스 진단 (인지/고려/전환)
- 요일/시간대별 성과 패턴 (데이터가 있다면)
- 다음 달 콘텐츠 캘린더 추천"
```
