# 콘텐츠 전환 분석 프레임워크

## 분석 구조

### 1단계: 콘텐츠 유형별 성과
- 릴스 / 캐러셀 / 단일이미지 / 스토리하이라이트
- 유형별 평균 도달, 인게이지먼트율, 전환율

### 2단계: 전환 퍼널
도달 → 좋아요/저장 → 링크클릭 → 자사몰유입 → 주문
- 각 단계별 전환율 (= 다음 단계 / 이전 단계)
- 유형별 퍼널 모양이 다름 (릴스: 도달 넓고 전환 좁음, 캐러셀: 도달 좁지만 전환 깊음)

### 3단계: 광고 vs 오가닉
- ROAS = 주문금액 / 광고비
- 오가닉 콘텐츠의 전환 효율 vs 광고 콘텐츠
- 최적 광고 예산 산출

### 4단계: 콘텐츠 → 매출 연결
- 게시일 전후 자사몰 주문 변화
- 주제/카테고리별 구매 전환 차이
- 제작자별 콘텐츠 ROI

## 핵심 지표

| 지표 | 산식 | 의미 |
|------|------|------|
| 전환율 | 주문건수/도달수 | 콘텐츠의 최종 구매 유도력 |
| ROAS | 주문금액/광고비 | 광고 1원당 매출 |
| CPO | 광고비/주문건수 | 주문 1건 획득 비용 |
| 저장율 | 저장/도달수 | 콘텐츠 재방문 가치 |
| 인게이지먼트율 | (좋아요+댓글+저장+공유)/도달수 | 콘텐츠 품질 종합 |
