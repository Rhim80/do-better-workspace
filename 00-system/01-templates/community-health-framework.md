# 커뮤니티 건강도 분석 프레임워크

> 기반: Cohort Retention (Mixpanel), NPS (Reichheld, 2003), Community Flywheel

---

## 1. 코호트 리텐션 (Cohort Retention)

**출처**: Mixpanel Analytics Framework / SaaS 업계 표준 지표
**적용**: 시즌제 커뮤니티의 재등록률을 코호트(가입 시점)별로 추적

**핵심**: 언제 가입한 멤버가 가장 오래 남는가?

### 리텐션 KPI

| KPI | 공식 | 의미 |
|-----|------|------|
| 시즌 재등록률 | 재등록수 / 전시즌 등록수 x 100 | 다음 시즌 잔존율 |
| N시즌 생존율 | N시즌 연속 등록수 / 최초 등록수 x 100 | 장기 충성도 |
| 이탈 시점 | 가장 많이 이탈하는 시즌 차수 | 위험 구간 파악 |
| 충성 멤버 비율 | 3시즌+ 연속 멤버 / 전체 x 100 | 커뮤니티 안정성 |

### 리텐션 곡선 해석

```
시즌 1 → 2: 급격한 하락이 정상 (40~60% 잔존)
시즌 2 → 3: 완만해지면 건강 (70~80% 잔존)
시즌 3+: 안정기 (80%+ 유지)

경고 신호:
- 시즌 1→2 잔존율 40% 미만: 온보딩 문제
- 시즌 2→3 잔존율 60% 미만: 가치 전달 문제
- 충성 멤버 이탈 증가: 커뮤니티 방향성 문제 (가장 위험)
```

## 2. NPS (Net Promoter Score)

**출처**: Reichheld, F. (2003). "The One Number You Need to Grow". Harvard Business Review

**핵심**: "이 커뮤니티를 지인에게 추천하시겠습니까?" (0~10점)

### NPS 계산

```
추천자 (Promoter): 9~10점
중립자 (Passive): 7~8점
비추천자 (Detractor): 0~6점

NPS = 추천자 비율(%) - 비추천자 비율(%)

해석:
- NPS 50+: 최상급 (Apple, Netflix 수준)
- NPS 30~50: 우수
- NPS 10~30: 양호
- NPS 0~10: 주의
- NPS 음수: 위기
```

### 커뮤니티 NPS 특수성

- B2C 제품 대비 커뮤니티 NPS는 높게 나오는 경향 (자기 선택 편향)
- NPS 자체보다 **시즌 간 NPS 추이**가 중요
- 비추천자의 **구체적 사유**가 개선의 핵심

## 3. 만족도 매트릭스

**출처**: Importance-Performance Analysis (IPA), Martilla & James (1977)

**핵심**: 무엇이 중요하고, 무엇이 잘 되고 있는가?

### 2x2 매트릭스

```
                    만족도 높음
                        │
    유지 강화            │         차별화 포인트
    (잘하고 있음)         │         (강점 극대화)
                        │
  ──────────────────────┼──────────────────────
                        │
    저우선순위           │         긴급 개선
    (큰 영향 없음)       │         (불만 + 중요)
                        │
                    만족도 낮음

        중요도 낮음 ←──────→ 중요도 높음
```

### 커뮤니티 만족도 항목

| 항목 | 측정 | 중요도 판단 기준 |
|------|------|----------------|
| 팀 활동 | 팀활동만족도 | 재등록 의향과의 상관관계 |
| 이벤트 | 이벤트만족도 | 참석률 + 만족도 교차 |
| 네트워킹 | 네트워킹만족도 | NPS와의 상관관계 |
| 공간 | 공간만족도 | 피드백 언급 빈도 |
| 가성비 | 가성비만족도 | 이탈자와의 상관관계 |

## 4. 유입-이탈 퍼널

**출처**: AARRR (Pirate Metrics), McClure, D. (2007)

**핵심**: 어디서 유입되고, 어디서 이탈하는가?

### 커뮤니티 퍼널

```
인지 (SNS, 기사, 입소문)
  └→ 관심 (홈페이지 방문, 설명회)
      └→ 등록 (결제)
          └→ 활성화 (첫 팀 미팅 참석)
              └→ 리텐션 (재등록)
                  └→ 추천 (NPS 9~10, 지인 소개)
```

### 가입 경로별 전환율

각 가입 경로(인스타/지인추천/기사/검색 등)별로:
- 등록 전환율
- 첫 시즌 활성도
- 재등록률
- NPS

을 비교하면 **가장 효율적인 유입 채널**을 식별할 수 있음.

> 지인 추천 유입이 보통 재등록률이 가장 높음 (기대치 매칭이 정확하기 때문)
