# 채널별 수익성 분석 프레임워크

## 분석 구조

### 1단계: 채널별 매출/수익 구조
- 채널별 총 매출 (주문금액 합계)
- 채널별 수수료 합계
- 채널별 원가 합계
- 채널별 영업이익 = 매출 - 수수료 - 원가
- 채널별 영업이익률 = 영업이익 / 매출

### 2단계: 상품 x 채널 매트릭스
- 상품별 채널 수익성 교차표
- "이 상품을 이 채널에서 팔면 얼마 남는가"
- 마이너스 마진 상품-채널 조합 식별

### 3단계: 시계열 분석
- 월별 채널 매출 추이
- 시즌성 패턴 (설날, 발렌타인, 화이트데이)
- 채널 의존도 변화

### 4단계: 의사결정 매트릭스
- 채널별 Keep / Scale / Drop 판단
- 기준: 이익률, 성장세, 전략적 가치 (브랜드 노출 등)

## 핵심 지표

| 지표 | 산식 | 의미 |
|------|------|------|
| 채널 영업이익률 | (매출-수수료-원가)/매출 | 채널의 실질 수익성 |
| 건당 영업이익 | 영업이익/주문건수 | 주문 1건의 가치 |
| 채널 집중도 | 채널매출/전체매출 | 특정 채널 의존 리스크 |
| 반품/취소율 | 반품+취소건/전체건 | 채널별 운영 비용 지표 |
