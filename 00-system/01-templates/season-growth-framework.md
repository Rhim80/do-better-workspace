# 시즌 성장 전략 프레임워크

> 기반: Growth Equation, Ansoff Matrix (1957), Unit Economics

---

## 1. 성장 방정식 (Growth Equation)

**핵심**: 시즌 등록수 = 전시즌 재등록 + 신규 유입 - 이탈

### 분해 공식

```
다음 시즌 등록수 = (현 시즌 등록수 x 재등록률) + 신규 유입수

예시:
현재: 152명, 재등록률 65%, 목표: 200명
재등록 예상: 152 x 0.65 = 99명
필요 신규: 200 - 99 = 101명

레버 1: 재등록률 65% → 75%로 올리면
  재등록: 152 x 0.75 = 114명, 필요 신규: 86명

레버 2: 신규 유입 채널 확대
  현재 신규 42명 → 86~101명 필요 (2~2.4배)
```

### 레버별 난이도

| 레버 | 효과 | 난이도 | 비용 |
|------|------|--------|------|
| 재등록률 +10%p | 높음 | 중 (기존 멤버 관리) | 낮음 |
| 신규 유입 2배 | 높음 | 상 (마케팅 확대) | 높음 |
| 이탈 방지 (환불 감소) | 낮음 | 하 (소수) | 낮음 |
| 가격 조정 | 중간 | 상 (브랜드 영향) | 리스크 |

## 2. 안소프 매트릭스 (Ansoff Matrix)

**출처**: Ansoff, H.I. (1957). "Strategies for Diversification". Harvard Business Review

**핵심**: 성장 방향을 4가지로 분류

```
              기존 서비스          신규 서비스
           ┌─────────────────┬─────────────────┐
기존 고객  │  시장 침투        │  서비스 개발     │
           │  (재등록률 강화)  │  (새 팀/이벤트)  │
           ├─────────────────┼─────────────────┤
신규 고객  │  시장 개발        │  다각화          │
           │  (새 채널 유입)   │  (새 포맷/가격)  │
           └─────────────────┴─────────────────┘
```

### 커뮤니티 적용

**시장 침투** (가장 안전):
- 재등록 멤버 혜택 강화
- 파트너 시스템 활성화
- 기존 멤버 추천 프로그램

**서비스 개발** (중간 리스크):
- 새로운 팀 주제 (트렌드 반영)
- 이벤트 포맷 다양화
- 온라인/하이브리드 옵션

**시장 개발** (높은 투자):
- 새로운 유입 채널 (유튜브, 뉴스레터)
- 타깃 확장 (주니어, 시니어)
- 지역 확장 (서울 외)

**다각화** (가장 리스크):
- 기업 대상 B2B 프로그램
- 해외 커뮤니티
- 유료 콘텐츠 플랫폼

## 3. 유닛 이코노믹스 (Unit Economics)

**핵심**: 멤버 1명당 경제성

### 기본 지표

| 지표 | 공식 | 의미 |
|------|------|------|
| ARPM | 총매출 / 등록수 | 멤버당 평균 매출 |
| CAC | 마케팅비 / 신규 등록수 | 신규 멤버 획득 비용 |
| LTV | ARPM x 평균 재등록 횟수 | 멤버 생애 가치 |
| LTV/CAC | LTV / CAC | 3x 이상이면 건강 |
| 멤버당 비용 | 총비용 / 등록수 | 멤버 1명 서빙 비용 |
| 멤버당 이익 | 영업이익 / 등록수 | 멤버 1명당 이익 |

### 시즌별 비용 구조

```
고정비 (등록수 무관):
- 공간 임대료
- 운영 인건비
- 기본 이벤트 비용

변동비 (등록수 비례):
- 이벤트 F&B
- 교재/핸드아웃
- 해외 필드트립 (있는 시즌만)

손익분기점 = 고정비 / (멤버십 가격 - 멤버당 변동비)
```

## 4. 이벤트 ROI

**핵심**: 어떤 이벤트가 가성비가 좋은가?

### 이벤트 효과 측정

| 지표 | 공식 | 활용 |
|------|------|------|
| 참석당 비용 | 이벤트 비용 / 참석수 | 비용 효율성 |
| 만족도/비용 | 만족도 평균 / (비용/참석수) | 투자 대비 만족 |
| 신청 경쟁률 | 신청수 / 정원 | 수요 파악 |
| 노쇼율 | 노쇼수 / 신청수 x 100 | 실행 효율성 |

### 이벤트 유형별 벤치마크

```
정기 이벤트 (오프닝/클로징):
- 참석률 80%+ 목표
- 만족도 4.0+ 목표
- 커뮤니티 소속감 강화 역할

특별 이벤트 (필드트립/캠핑):
- 높은 비용, 높은 만족도
- 마케팅 콘텐츠 생산 가능
- 신규 유입 트리거

번개 모임:
- 비용 거의 없음
- 만족도 높음 (자발적 참여)
- 네트워킹 가치 극대화
```

> 비용 대비 만족도 최적점은 보통 "저비용 + 자발적 참여" 포맷에서 나옴
