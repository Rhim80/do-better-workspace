# 인플루언서 효과 분석 프레임워크

> 기반: Influencer Marketing ROI, Engagement Rate Benchmarks, Earned Media Value (EMV)

---

## 1. 핵심 성과 지표

**출처**: Influencer Marketing Hub (2024), CreatorIQ Industry Benchmarks

### 기본 KPI

| KPI | 공식 | 의미 |
|-----|------|------|
| Engagement Rate (ER) | (좋아요+댓글+저장+공유) / 노출수 x 100 | 콘텐츠 반응도 |
| 인플루언서 ROI | 매출기여 / 비용 | 투자 대비 수익 |
| CPA | 비용 / 전환수 | 전환 1건당 비용 |
| CPE | 비용 / (좋아요+댓글+저장+공유) | 반응 1건당 비용 |
| Link CTR | 링크클릭 / 노출수 x 100 | 행동 전환율 |

### 심화 KPI

| KPI | 공식 | 의미 |
|-----|------|------|
| EMV (Earned Media Value) | 노출수 x CPM단가/1000 + 인게이지먼트 x 가중치 | 미디어 환산 가치 |
| 전환당 매출 | 매출기여 / 전환수 | 구매 품질 (객단가) |
| 콘텐츠 수명 | 게시 후 engagement 50% 감소까지 일수 | 콘텐츠 지속력 |
| Save Rate | 저장수 / 노출수 x 100 | 구매 고려 지표 (저장 = 관심) |

## 2. 등급별 벤치마크

**출처**: HypeAuditor State of Influencer Marketing 2024, Influencer Marketing Hub

### 팔로워 기준 등급 분류

| 등급 | 팔로워 수 | ER 벤치마크 | 특성 |
|------|-----------|-------------|------|
| 메가 | 100만+ | 1~3% | 인지도 극대화, 비용 높음, 전환 효율 낮음 |
| 매크로 | 10~100만 | 3~6% | 도달+전환 균형, 카테고리 전문성 |
| 마이크로 | 1~10만 | 5~12% | 전환 효율 최고, 타깃 정합성 높음 |
| 나노 | 1만 미만 | 8~18% | 커뮤니티 신뢰 높음, 스케일 제한 |

### 등급별 비용 대비 효율 패턴

```
비용 효율 (ROI)
높 │  ★ 나노        ★ 마이크로
   │
   │           ★ 매크로
   │
   │                          ★ 메가
낮 │
   └──────────────────────────────
   적음                     많음
              도달 범위 (Reach)

핵심 인사이트:
- 메가는 "인지도 캠페인"에 적합 (ROAS 낮지만 도달 크다)
- 마이크로는 "전환 캠페인"에 적합 (ROAS 높고 타깃 정합성)
- "1명의 메가 vs 10명의 마이크로" → 대부분 마이크로가 총전환 우세
```

## 3. 플랫폼별 분석

**출처**: Sprout Social Index 2024, Later Social Media Benchmarks

### 플랫폼별 특성

| 플랫폼 | 강점 | ER 벤치마크 | 콘텐츠 포맷 |
|--------|------|-------------|-------------|
| 인스타그램 | 비주얼+쇼핑 연동 | 3~6% | 릴스, 피드, 스토리 |
| 틱톡 | 도달 폭발, MZ 집중 | 5~10% | 쇼츠, 라이브 |
| 유튜브 | 장기 콘텐츠, SEO | 2~5% | 쇼츠, 리뷰 영상 |

### 플랫폼-목적 매칭

```
인지도 확대    → 틱톡 (알고리즘 도달), 유튜브 (검색 노출)
구매 고려      → 인스타그램 (쇼핑태그, 저장), 유튜브 (리뷰)
즉시 전환      → 인스타그램 (스와이프업, 쇼핑), 틱톡 (TikTok Shop)
브랜드 스토리  → 유튜브 (롱폼), 인스타그램 (카루셀)
```

## 4. 콘텐츠 유형별 분석

### 유형별 성과 기대

| 유형 | 도달 | ER | 전환 | 수명 |
|------|------|-----|------|------|
| 릴스/쇼츠 | 매우 높음 | 높음 | 중간 | 짧음 (3~7일) |
| 피드 | 중간 | 중간 | 높음 | 중간 (7~14일) |
| 스토리 | 낮음 | 낮음 | 중간 | 매우 짧음 (24시간) |
| 라이브 | 중간 | 매우 높음 | 높음 | 실시간 한정 |

### 카테고리-유형 최적 조합

```
아우터/의류  → 릴스 (착장 영상) + 피드 (스타일링 사진)
스니커즈     → 쇼츠 (언박싱) + 라이브 (발매 이벤트)
뷰티        → 릴스 (튜토리얼) + 스토리 (일상 사용)
라이프스타일 → 피드 (무드보드) + 카루셀 (큐레이션)
```

## 5. 인플루언서 티어 매트릭스

### 의사결정 프레임워크

```
                비용 효율 (ROI)
              높음              낮음
         +----------------+----------------+
    도   |  Sweet Spot     |  Volume Play   |
    달   | (마이크로)       | (메가)         |
    범   | → 전환 캠페인    | → 인지도 캠페인 |
    위   | 비중: 40~50%     | 비중: 10~15%   |
    넓   +----------------+----------------+
    음   |  Hidden Gem     |  Reconsider    |
         | (나노)          | (저성과 매크로) |
    좁   | → 니치 타깃      | → 크리에이티브  |
    음   | 비중: 15~20%     | 개선 or 중단   |
         +----------------+----------------+

인플루언서 포트폴리오 권장 배분:
- 마이크로 (40~50%): 전환의 핵심 엔진
- 매크로 (20~30%): 도달+전환 밸런스
- 메가 (10~15%): 시즌 캠페인, 브랜드 임팩트
- 나노 (10~20%): 커뮤니티 침투, UGC 생성
```

## 6. EMV (Earned Media Value) 계산

**출처**: Ayzenberg (2015), Tribe Dynamics methodology

```
EMV = 미디어 노출 가치 + 인게이지먼트 가치

미디어 노출 가치 = 노출수 x (업종 평균 CPM / 1000)
인게이지먼트 가치 = 좋아요 x 0.1 + 댓글 x 1.0 + 저장 x 1.5 + 공유 x 2.0
(가중치는 행동의 깊이에 비례)

비교 활용:
- EMV vs 실제 집행 비용 → EMV가 높으면 비용 대비 가치 높음
- 인플루언서 간 EMV 비교 → 동일 비용 대비 미디어 가치 비교
```

- 인플루언서 티어 매트릭스 시각화 (버블차트: X축 비용, Y축 ROI, 버블 크기 노출수)
