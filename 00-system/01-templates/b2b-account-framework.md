# B2B 거래처 분석 프레임워크

> 기반: HHI (Herfindahl, 1950), Pareto/ABC (Juran, 1951), RFM (Hughes, 1994), OTIF (Gunasekaran, 2001)

---

## 1. 거래처 집중도 분석 (Customer Concentration)

**출처**: Herfindahl, O.C. (1950). *Concentration in the Steel Industry*. Columbia Univ. / Hirschman, A.O. (1964). "The Paternity of an Index." *American Economic Review*, 54(5), 761.
**배경**: 미국 법무부(DOJ)가 시장 경쟁도 측정에 사용하는 표준 지표. B2B 제조업에서는 거래처 의존도 리스크 평가에 활용.

**핵심**: 소수 거래처에 매출이 집중되어 있는가? 한 곳이 이탈하면 얼마나 위험한가?

### HHI (Herfindahl-Hirschman Index)

```
HHI = Σ (si)²

si = i번째 거래처 매출 / 전체 매출 x 100
```

| HHI 구간 | 해석 | 조치 |
|----------|------|------|
| < 1,500 | 분산형 (저위험) | 현행 유지 |
| 1,500 ~ 2,500 | 중간 집중 | 신규 거래처 확대 검토 |
| > 2,500 | 고집중 (고위험) | 의존도 분산 전략 필요 |

### 파레토(80/20) + 상위 N개 비중

```
상위 N개 거래처 집중도 = ΣTop-N 매출 / 전체 매출 x 100
```

**출처**: Pareto, V. (1896). *Cours d'economie politique*. / Juran, J.M. (1951). *Quality Control Handbook*. McGraw-Hill.

| 패턴 | 해석 |
|------|------|
| 상위 20% 거래처 = 80%+ 매출 | 전형적 파레토. A등급 집중 관리 |
| 상위 20% 거래처 = 60~70% 매출 | 건전한 분산. 균형 상태 |
| 상위 5개 거래처 = 50%+ 매출 | 특정 거래처 의존 위험. 이탈 시나리오 검토 필요 |

### 적용 질문

- 상위 5개 거래처가 전체 매출의 몇 %를 차지하는가?
- 최대 거래처가 이탈하면 매출 영향은?
- 거래처 집중도가 지난 분기 대비 높아졌는가, 낮아졌는가?

## 2. 거래처 유형별 수익성 (RFM 분석)

**출처**: Hughes, A.M. (1994). *Strategic Database Marketing*. Probus. / Holma, A. et al. (2018). "RFM for product-oriented services." *J. Management and Governance*. doi:10.1007/s10997-018-9447-3

**핵심**: 거래처를 Recency(최근성), Frequency(빈도), Monetary(금액)으로 등급화

### RFM 변수 정의

| 변수 | 정의 | B2B 제조업 적용 |
|------|------|----------------|
| R (Recency) | 마지막 발주로부터 경과 일수 | 거래처 활성/휴면 판단 |
| F (Frequency) | 분석 기간 내 발주 건수 | 정기 발주 vs 프로젝트 단발 |
| M (Monetary) | 분석 기간 내 총 발주 금액 | 건당 평균 = M / F |

### 거래처 유형별 RFM 해석

| 유형 | R | F | M | 해석 |
|------|---|---|---|------|
| 대리점 | 짧음 | 높음 | 중간 | 안정적 물량. 단가 협상력 관리 |
| 시공사 | 가변 | 낮음 | 매우 큼 | 프로젝트 단위. 건당 규모가 핵심 |
| 인테리어업체 | 중간 | 중간 | 중~높 | 프리미엄 제품 비중 높음 |
| 관공서 | 길음 | 낮음 | 큼 | 낙찰 기반. 예측 어려우나 안정적 |

### 5등급 분류

```
각 변수를 1~5 등급으로 나눈 후 조합:

555 (VIP) = 최근 발주 + 자주 + 대량 → 핵심 관리 대상
511 (이탈 위험) = 최근 발주했지만 빈도/금액 낮음 → 성장 기회 or 이탈 선행
155 (휴면 VIP) = 과거 대량/빈번했으나 최근 없음 → 긴급 관계 회복
```

### 적용 질문

- 거래처 유형별 건당 평균 발주액은 얼마인가?
- 시공사 발주가 전체 매출에서 차지하는 비중은? 계절별 변동은?
- 3개월 이상 발주가 없는 기존 거래처는 몇 곳인가?

## 3. 지역별 수요 분석 (Geographic Demand)

**출처**: Ellison, G. & Glaeser, E. (1997). "Geographic Concentration in U.S. Manufacturing Industries." *Journal of Political Economy*, 105(5), 889-927.
**배경**: 지역별 매출 집중도를 파악하여 영업 자원 배분과 물류 최적화에 활용

### 지역 집중도 공식

```
지역별 매출 비중 = 해당 지역 매출 / 전체 매출 x 100

지역 HHI = Σ (지역별 매출 비중)²
```

### 해석 가이드

| 패턴 | 해석 | 조치 |
|------|------|------|
| 단일 지역 40%+ | 지역 의존도 높음 | 타 지역 영업 확대 |
| 수도권 집중 60%+ | 건설/인테리어 시장 특성 반영 | 신도시 물량 추적 |
| 특정 지역 급증 | 대형 프로젝트 영향 | 일회성 vs 지속 판단 |

### 적용 질문

- 수도권 vs 비수도권 매출 비중은?
- 특정 지역 매출 급증은 단일 프로젝트 효과인가, 구조적 변화인가?
- 지역별 거래처 수 대비 매출 효율은? (거래처당 평균 매출)

## 4. 발주 패턴 분석 (Order Pattern & OTIF)

**출처**: Gunasekaran, A., Patel, C. & Tirtiroglu, E. (2001). "Performance Measures in a Supply Chain Environment." *IJOPM*, 21(1/2), 71-87. / Gunasekaran, A., Patel, C. & McGaughey, R.E. (2004). *IJPE*, 87(3), 333-347.

**핵심**: 발주 시기, 납품 이행률, 계절성을 분석하여 생산/물류 계획 최적화

### OTIF (On-Time In-Full)

```
OTIF = (정시 + 완전 납품 건수) / 전체 발주 건수 x 100

납기 준수율 = 약정일 내 납품 건수 / 전체 납품 건수 x 100
```

| OTIF | 해석 |
|------|------|
| 95%+ | 우수. 현행 유지 |
| 90~95% | 보통. 지연 원인 분석 |
| 90% 미만 | 개선 필요. 생산/물류 병목 확인 |

### 계절성 지수 (Seasonal Concentration)

**출처**: Hylleberg, S. (1992). *Modelling Seasonality*. Oxford University Press.

```
월별 계절성 지수 = 해당 월 발주액 / 연간 월평균 발주액

지수 > 1.5 → 계절 집중 발생. 생산/재고 완충 필요
지수 < 0.7 → 비수기. 프로모션 또는 생산 조정 검토
```

### 적용 질문

- 납기 지연이 발생한 건수와 원인은? (생산 지연 / 물류 / 자재 부족)
- 발주가 특정 주(week)에 몰리는 패턴이 있는가?
- 봄 이사철(3~5월) 발주 증가율은 전년 대비 어떠한가?
- 발주 취소 건의 공통 원인은?

## 5. 종합 분석: 거래처 포트폴리오 매트릭스

### 2x2 매트릭스 (매출 규모 x 성장률)

```
              매출 규모 큼          매출 규모 작음
성장률 높음 │  Star (핵심 성장)    │  Question Mark (관찰)  │
성장률 낮음 │  Cash Cow (안정)     │  Dog (재검토)          │
```

**출처**: BCG Growth-Share Matrix (Henderson, 1970) 원리를 거래처 포트폴리오에 적용

### 거래처별 적용

| 포지션 | 전략 |
|--------|------|
| Star | 관계 심화, 전담 영업, 우선 납품 |
| Cash Cow | 효율적 관리, 마진 유지, 자동화 |
| Question Mark | 성장 잠재력 평가, 선별적 투자 |
| Dog | 비용 대비 수익 재검토, 조건 재협상 |

## CEO 보고 시 활용

경영진에게 보고할 때 이 프레임워크의 핵심 지표를 다음 순서로 제시:

```
1. 전체 매출 + 거래처 수 (스케일)
2. 상위 5개 거래처 집중도 + HHI (리스크)
3. 거래처 유형별 매출/건당 규모 (구조)
4. 지역별 분포 (기회)
5. OTIF + 납기 이슈 (운영 품질)
6. 봄 이사철 시뮬레이션 (액션)
```
