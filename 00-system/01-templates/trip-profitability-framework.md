## 1. 트립 단위 경제학 (Trip Unit Economics)

**출처**: Gupta, S. & Lehmann, D. (2006). Managing Customers as Investments. Wharton School Publishing

**핵심**: ��립 1회의 수익성을 직접 매출뿐 아니라 후속 가치까지 포함하여 측정

### 기본 공식

```
영업이익 = 총매출 - 총비용
영업이익률 = 영업이익 / 총매출 x 100

비용 구성:
  항공비 + 숙박비 + 현지교통비 + 식음료비 + 가이드비 + 기타
```

### B2B vs B2C 비교 기준

| 지표 | B2B | B2C |
|------|-----|-----|
| 매출 구조 | 일괄 계약 (기업 단가) | 1인 참가비 x 인원 |
| 원가 | 유사 (인원 비례) | 유사 |
| 이익률 기대치 | 높음 (프리미엄 단가) | 중간 |
| 후속 가치 | 높음 (컨설팅 전환) | 낮~중간 |

## 2. 충원율 (Capacity Utilization)

**핵심**: 정원 대비 실제 참가 비율. 고정비(항공, 가이드)는 인원과 무관하므로 충원율이 이익률을 직접 좌우

```
충원율 = 참가자수 / 정원 x 100

손익분기 참가자수 = 총비용 / 1인 참가비
```

### 해석

| 충원율 | 의미 | 액션 |
|--------|------|------|
| 90%+ | 건강 | 정원 확대 검토 |
| 70-90% | 양호 | 현행 유지 |
| 50-70% | 주의 | 마케팅 강화 또는 정원 축소 |
| 50% 미만 | 위험 | 트립 존폐 검토 |

## 3. 참가자 ��로필 분석

### 유입경로별 분석

유입��로는 콘텐츠 전략과 직결:
- **뉴스레터/인스타**: 자사 콘텐츠 효과
- **지인추천**: 만족도 기반 구전
- **도서/강연**: 브랜드 자산 효과
- **기업**: B2B 영업

### 만족도-재참가 매트릭스

| | 재참가 의향 높음 | 재참가 의향 낮음 |
|---|-----------------|-----------------|
| **만족도 높음** | 로열 고객 (VIP 관리) | 1회성 만족 (재방문 유인 설계) |
| **만족도 낮음** | 기대 불일치 (개선 필요) | 이탈 위험 (원인 분석) |

## 4. 도시별 비교 분석

### 비교 축

| ��� | 지표 |
|------|------|
| 수익성 | 영업이익률 |
| 수요 | 충원율 + 트립 문의 건수 |
| 만족도 | 평균 만족도 + NPS |
| 후속 가치 | 컨설팅 전환 건수/금액 |

### 의사결정 로직

```
수익성 높고 + 수요 높음 → 확대 (편수 증가)
수익성 높고 + 수요 낮음 → 마케팅 강화
수익성 낮고 + 수요 높음 → 원가 구조 개선
수익성 낮고 + 수요 낮음 → 축소 또는 중단
```
